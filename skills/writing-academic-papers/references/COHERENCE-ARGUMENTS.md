## 1. 一貫性（Coherence）の計画

### 計画（Planning）とは

計画とは、課題に効果的に応答するために情報とアイデアを選択・整理するプロセス。

**手順:**
1. 課題の文言を分析（"unpacking"）し、期待される内容を理解
2. アウトライン（outline）を作成

### アウトラインの要件

- 単なるトピックリストではない
- **全体と各セクションに明確な組織化原則（organizing principle）が必要**

| 組織化原則 | 説明 |
|---------|------|
| **時系列順** | 出来事や発展を時間順に配置 |
| **重要度順** | 最も重要な要素から順に提示 |
| **General→Specific** | 一般的な情報から具体的な情報へ展開 |

### 学術論文における主要な組織化フレームワーク

| フレームワーク | 内容 | 適用例 |
|-------------|------|--------|
| **Classification** | 対象をグループ、クラス、カテゴリに分類 | 「学習困難の種類」 |
| **Comparison and Contrast** | 類似点と相違点の検討 | 「二つの教育アプローチの比較」 |
| **Cause and Effect** | 原因と結果の関係、影響の分析 | 「貧困が学習成果に及ぼす影響」 |
| **Problem-Solution** | 問題の記述・分析と解決策の評価 | 「教室での行動管理の課題と対処法」 |

**補足要素:**
- **Description**（記述）
- **Narrative**（物語的展開）
- **Definition**（用語定義）
- **Exemplification**（例示）

これらの要素は重複・併用される（例: Problem-Solution では Cause and Effect も議論される）。

## 2. 編集と推敲

### ライティングコミュニティの真実

- **すべてのレベルのライターが編集と推敲を繰り返す**
- 初心者も熟練者も、明確で一貫性のある文章を作るには努力が必要
- 経験豊富なライターほど編集・推敲に時間をかける（研究結果）

### 読者の視点で読む（Putting yourself in the reader's shoes）

執筆中も執筆後も、頻繁に自分のテキストを読み返し、以下を自問すること:

| チェック項目 | 確認内容 |
|---------|--------|
| **理解可能性** | 他者にとって意味が通じるか |
| **組織化原則** | 各ポイント・アイデアが明確な原則に従って整理されているか |
| **論理的順序** | 提示順序は論理的か |
| **明示的なつながり** | アイデア間のリンクが明示されているか |
| **曖昧さの有無** | 曖昧または不明瞭な箇所はないか |
| **流れ（Flow）** | テキスト全体が自然に流れるか |

### 編集時の焦点

| ステージ | 焦点 |
|--------|------|
| **編集段階** | **意味（meaning）** に集中 |
| **最終段階** | 表面的な文法・句読点のエラーを修正（proofreading） |

**重要:** 締切直前に完成させるのではなく、時間をかけて複数回読み直すことが必須。

## 3. イントロダクションと結論

### イントロダクションの構成要素

| 要素 | 内容 |
|-----|------|
| **目的の明示** | エッセイのタイトル・問いに直接言及 |
| **背景情報** | 主要概念、定義、文脈の説明 |
| **展開方法の提示** | エッセイの構成を簡潔に概観（overview） |

**イントロダクションの例テンプレート:**

```
[背景情報と主張の紹介]

**The aim of this essay is to** [目的].
**It will begin by** [第1セクション].
**It will then** [第2セクション].
**It will go on to** [第3セクション].
```

### 典型的なシグナルフレーズ（Signposts）

| フレーズ | 後続する動詞形 | 説明 |
|--------|------------|------|
| The aim of this essay is | **to do** | 目的を示す |
| It will begin by | **doing** | 最初のセクション |
| It will then | **outline/examine/analyse** | 次のセクション |
| It will go on to | **do** | さらなる展開 |

### エッセイで使用する動詞の選択

| 動詞 | 意味・深さ |
|-----|---------|
| outline, describe, identify | 提示・記述レベル |
| examine, look into, analyse | より深い分析 |
| evaluate, explore | 批判的評価・探究 |

**注意:** これらの動詞は互換ではない。エッセイタイトルの指示動詞と整合させること。

### 結論（Conclusion）の構成要素

| 要素 | 内容 |
|-----|------|
| **イントロダクションへの参照** | 目的達成を示す |
| **論点の要約** | 本論で議論した主要ポイントをまとめる |
| **自分の立場の明示** | トピックに対する自身の見解とその意義 |
| **実践的な提言** | 実践への示唆・推奨事項 |

## 4. 結束性（Cohesion）と段落構造

### 効果的な段落の特徴

| 特徴 | 説明 |
|-----|------|
| **単一の焦点** | 一つの統一されたポイントを扱い、脱線しない |
| **トピックセンテンス** | 第1文で主題を導入（多くの場合） |
| **General→Specific** | 一般的な記述から具体的な情報へ展開 |
| **論理的接続** | アイデアを論理的に順序立てて接続 |

### General と Specific の関係

| 段落タイプ | General（導入） | Specific（詳細） |
|---------|--------------|----------------|
| 概念定義型 | 重要概念・用語の導入 | 関連性、重要性、分析、根拠 |
| 主張型 | 一般的な主張・声明 | 説明、分類、論証 |
| 推奨型 | 推奨事項の提示 | 説明、例示、個別ポイントの展開 |

**Topic Sentence（トピックセンテンス）** = 段落全体のテーマを確立する一般的な記述（"umbrella statement" とも呼ばれる）

### Old と New 情報の配置

効果的な段落展開の原則:
- **Old（既知）情報への参照** → **New（新規）情報の提示**
- 前の文や既出の情報に言及してから、新しい情報を導入
- トピックとのつながりを保ち、新情報に焦点を当てる

**例:**

```
It is important to consider whether boys perform as well as girls in other subjects.
Looking at the Trends in International Mathematics and Science Study,
[old: boys' performance] boys marginally outperformed girls in mathematics.
[new: science results] In science, the results were broadly similar.
```

## 5. 参照表現と接続表現

### 参照表現（Referring back）の手法

#### 1. 繰り返し（Repetition）

- **良い繰り返し:** キーターム・概念を繰り返し、読者に主題を常起させる
- **悪い繰り返し:** 冗長な情報、同じ語の不自然な連続使用

**関連語チェーン（Word Chains）:**
同一カテゴリの関連語を使用（例: vocabulary → words → lexicon → names）

#### 2. 代名詞（Pronoun）の使用

| 判断基準 | 推奨アプローチ |
|--------|------------|
| 参照先が明確 | 代名詞（it, they, them, etc.）を使用可 |
| 参照先が曖昧 | 名詞を繰り返す |
| 専門用語 | 変更しない（例: "active listening" を "careful listening" に置換しない） |

#### 3. 同義語（Synonyms）の注意

| 状況 | 対処法 |
|-----|--------|
| 同義語の意味が不確実 | 元の語を繰り返す |
| 専門用語 | 絶対に変更しない |
| 慣用表現 | 従来の用法に従う |

### Back-Referencing に使う語句

| 語句 | 使用例 | 機能 |
|-----|--------|------|
| **this / these** | This way of looking at language... | 前の文・概念全体を指す |
| **that / those** | that development | 既出の名詞を指す |
| **the** | the findings | 既出の特定の対象 |
| **such** | such work | 前述の活動・取り組み |

### Signalling Nouns（シグナル名詞）

前の文を要約し、書き手の態度を示す名詞:

| 頻出Signalling Nouns | 教育分野頻出 |
|-------------------|-----------|
| time, case, point, view, period, process, approach, question, problem, area | review, programme, unit, scale, assignment, theme, curriculum, institution, survey, training, treatment |

**態度を示す例:**
- "this problem" vs "this challenge" （前者はネガティブ、後者はポジティブ）
- "this serious problem" vs "this minor issue"

### 接続表現（Linking Words and Phrases）

| 機能 | 接続表現 |
|-----|--------|
| **追加（Addition）** | in addition to, also, moreover, furthermore |
| **例示（Exemplification）** | for example, for instance |
| **目的（Purpose）** | in order to, so as to |
| **対比（Contrast）** | while, however, despite, though, although, nevertheless, on the contrary, in contrast |
| **理由（Reason）** | because, since, as, given |
| **結果（Result）** | thus, therefore, consequently, hence |
| **条件（Condition）** | given, if, provided that |

**重要:** これらの接続表現は**意味に基づいて**選択すること。装飾目的で使用しない。

### Signposts（道標表現）

読者に対して明示的に展開を示す表現:

| 用途 | パターン | 例 |
|-----|--------|---|
| **順序の提示** | One aspect ... Another aspect ... | One aspect of active listening is verbal communication. Another important element is non-verbal communication. |
| **エッセイ構成の提示** | The aim ... It will begin by ... It will then ... | （前述のイントロダクション例を参照） |

## 6. 論証の展開と批判的表現

### 一貫性のある論証の構築

**原則:** 読者の視点に立ち、読者が何を知りたいかを予測する

| ステップ | 内容 |
|--------|------|
| **読者の疑問を予測** | "Why?", "How?", "What evidence?" など |
| **明確な道標を提供** | 読者が迷わないようガイド |
| **厳しい読者を想定** | 納得させるために十分な根拠を提示 |

### 批判的言語表現（Language of Criticality）

批判性（criticality）と立場（stance）を示す言語的手段:

#### 1. 確信度を示す表現

| 確信度 | 表現例 |
|--------|--------|
| **確実** | will, is, clearly, evidently |
| **ほぼ確実** | form a strong argument, there is considerable evidence |
| **可能性** | may, can, often, many, some, possibly |
| **慎重** | to some extent, largely, reported, suggested |

**例:**
- "This approach **will** limit the negative effects." （確実）
- "This approach **may** limit the negative effects." （不確実）

#### 2. 程度を示す表現

| 程度 | 表現 |
|-----|------|
| 完全 | all, entirely, completely |
| 大部分 | largely, to a great extent, mainly |
| 部分的 | to some extent, partially, a number of |
| 小さい | marginally, slightly, minor |

#### 3. 評価を示す語彙選択

| ポジティブ | ネガティブ |
|---------|---------|
| innovative, enhance, extend, meaningful development, challenge | problems, struggle, not acceptable, complicated, serious problems, largely untested |

#### 4. 他者の見解を報告する動詞

| 報告動詞 | 書き手の態度 |
|--------|-----------|
| **argues** | 説得力のある根拠を持つ主張と評価 |
| **suggests** | 弱い推奨・提案 |
| **claims** | 疑問の余地がある主張 |
| **maintains** | 多数意見や反証に反して主張し続ける |
| **advocates** | 強く推奨 |
| **discourages** | 推奨しない |
| **found that** | 研究結果として確認 |
| **has suggested** | 示唆している（やや控えめ） |

#### 5. コメント副詞（Comment Adverbs）

| 副詞 | 態度 |
|-----|------|
| **Crucially** | 非常に重要・ポジティブと見なす |
| **Importantly** | 重要性を強調 |
| **Worryingly** | 懸念を表明 |
| **Clearly** | 明白であると主張 |
| **Indeed** | 強調・同意 |

### 段落レベルでの論証展開パターン

#### パターン1: 賛成→反対→再賛成

```
[Paragraph 1: ポジティブな主張]
- トピックセンテンスで肯定的見解
- 公式文書や研究を引用
- ポジティブな評価語を使用（"clearly", "extending", "enriching", "strong argument"）

[Paragraph 2: 反対意見の提示]
- "Nevertheless" で転換
- 反論や問題点を提示
- ネガティブな評価語を使用（"struggle", "problems", "not acceptable"）

[Paragraph 3: 再度ポジティブな主張を強化]
- "However" で再転換
- 具体例と学習機会の提示
- ポジティブな評価語で結論（"enhance", "extend", "meaningful development"）
```

### 批判性を示す言語の総まとめ

| カテゴリ | 手法 | 例 |
|--------|------|---|
| **明示的意見** | clearly, there is a great deal to be said, can positively enhance, indeed | "Clearly this recognises possibilities for extending..." |
| **要約名詞句** | these bygone writers, such classic texts | 前述内容を解釈・評価しながら参照 |
| **報告動詞の選択** | found that, has suggested, argues, claims | 文献の信頼性・妥当性を評価 |
| **確信度の調整** | will/may, often, many, can, to some extent | 過度な一般化や根拠のない主張を避ける |

## まとめ

一貫性のある文章と説得力のある論証を構築するためには:

1. **計画段階で組織化原則を明確にする**
2. **編集・推敲を繰り返し、読者の視点でテキストを見直す**
3. **明確なイントロダクションと結論を提供する**
4. **General→Specific、Old→New の原則で段落を構造化する**
5. **効果的な参照表現・接続表現で結束性を高める**
6. **批判的言語表現を駆使して自身の立場を明確にする**

これらの原則を体系的に適用することで、読者にとって理解しやすく、説得力のあるアカデミックテキストを作成できる。
