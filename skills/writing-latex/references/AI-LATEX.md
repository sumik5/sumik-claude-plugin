# LaTeXでのAI活用ガイド

本ドキュメントは、AIツール（ChatGPT、Claude等の大規模言語モデル）をLaTeX作業に活用する方法をまとめたものです。

---

## AI (人工知能) とは

**定義**: ソフトウェアやマシンによってシミュレートされた知能

**動作原理**:

- 大量のデータ処理
- ロジック、統計、アルゴリズム的トレーニングによる学習

**Generative AI (生成AI)**:

- テキスト、画像、動画を生成
- LaTeXにおいては**テキスト生成**（コンテンツと LaTeX ソースコード）が有用

**LLM (Large Language Model)**:

- 膨大なデータセットで訓練された大規模言語モデル
- **プロンプト**（入力テキスト）に基づき、統計的関係性を利用して次の単語を予測

## AIツール使用の注意事項

### データプライバシー

**重要**: AIツールは入力データを保存・処理し、トレーニングや分析に使用する可能性がある。

- **機密情報をオンラインAIに入力しないこと**
- 企業秘密、個人情報、未公開研究データなどは避ける

### 最新情報の確認

- AIの学習データは特定時点までのもの（例: GPT-3.5は2022年1月まで）
- **最新パッケージの情報は必ずCTAN / texdoc / 公式ドキュメントで確認**

### コード検証の必要性

- **生成されたLaTeXコードは必ずコンパイルして検証**
- AIは論理的に正しく見えるが誤ったコードを生成することがある

### 複雑なレイアウトの限界

- 複雑なページレイアウトやカスタムデザインは、AIだけでは完全に実現できない場合がある

## LaTeXでのAI活用パターン

### 1. LaTeXに関する質問

**プロンプト例**:

- *LaTeXの利点は何ですか？*
- *XeLaTeXとLuaLaTeXの違いは？*
- *LaTeXで`\include`と`\input`のどちらを使うべきですか？*
- *Beamerでスライドを作成する方法は？*
- *このLaTeXコードのエラーはどこですか？* [コードを添付]

**フォローアップ**:

- *もっと詳しく*
- *具体例を見せて*
- *特定の部分を説明して*
- *ステップごとに分解して*
- *わかりにくい部分を明確にして*
- *もう一度試してコンテキストを追加*

**利点**:

- 膨大な知識ベースから即座に回答を得る
- 会話形式で深掘り可能
- コードスニペット付きの実例が得られる

### 2. コンテンツ改善

#### 校正と文法チェック

**プロンプトテンプレート**:

```
技術エディタとして振る舞ってください。
非公式だが冷静で技術的なトーンを維持し、過度な熱意を避けてください。
以下のテキストを言い換え、良い部分を保持しつつ全体の品質を向上させてください。

[あなたのテキスト]
```

**応答例**:

元のテキスト（Wikipedia）:
> *LaTeX uses the TeX typesetting program for formatting its output, and is itself written in the TeX macro language.*

AI改善版:
> *LaTeX formats its output using the TeX typesetting program and is written in the TeX macro language itself.*

**効果**:

- スペルと文法の修正
- 指定したトーンへの調整
- 同義語の提案でインパクトと明瞭さを向上

#### トーン調整

**デフォルトのトーン**: ChatGPTはしばしば熱狂的な表現を使う

**中立的トーンの要求例**:

```
箇条書きを使わずに短縮してください
```

**過度に興奮したトーンの実験例**:

```
同じことをもっと興奮して伝えて
→ [段階的に繰り返し]
→ 超絶クレイジーに興奮して！
```

**結果**: 誇張表現・比喩・感嘆詞を多用した文章が生成される（笑い話として）

#### ライターズブロックの克服

**活用法**:

- AIにアイデアやキーポイントを生成させる
- 導入文・概要・目次をドラフトさせる
- 箇条書きや表で構造化させる
- 文法を気にせず速記し、AIに修正させる
- 母語で書き、AIに翻訳させる

**例**:

```
プロンプト: 最も一般的な6種類の図を箇条書きで示し、それぞれを説明し、
どの科学分野やビジネス分野で最も人気があるかを教えてください

応答:
1. フローチャート:
   - 説明: プロセスやワークフローを視覚的に表現
   - 人気分野: ビジネス（プロセス最適化、プロジェクト管理）、科学（実験手順、データ分析ワークフロー）

[続く...]
```

**フォローアップ**:

```
各図タイプに対応するLaTeXパッケージを挙げてください
```

**結果**: 各図タイプのパッケージと完全なコード例が得られる

### 3. LaTeXコード生成

#### 基本的な使い方

1. **言語指定**: プロンプトまたは会話の早い段階でLaTeXを明示
2. **明確な要求**

```
プロンプト: m行n列の行列のLaTeXコードスニペットを生成してください。
a_ijエントリと点（...）を使用してください。
```

3. **説明を求める**

```
プロンプト: bmatrix環境は何をしていますか？
```

**応答**: 意味と構文の説明、必要なパッケージ（amsmath）の情報

4. **変更を要求**

```
プロンプト: 行列のセルで a の代わりに x を使用してください
プロンプト: 角括弧の代わりに丸括弧を区切り文字として使用してください
```

**結果**: `bmatrix`が`pmatrix`に変更される

5. **コピー＆ペースト**: **Copyボタン**を使うかコードをハイライトしてエディタに貼り付け
6. **コンパイル**: PDFで結果を確認

#### 高度な生成例

**テンプレート生成**:

```
プロンプト: 3セクション、各セクション3スライド、Singaporeテーマを使用した
目次付きLaTeX Beamerプレゼンテーションを作成してください
```

**グラフ・プロット**:

```
プロンプト: pgfplotsを使って正弦波と余弦波をプロットしてください
```

**TikZ図形**:

```
プロンプト: TikZでロボットを描いてください
```

**注意**: 創造的なタスクでは結果が面白いが不完全な場合がある

**BibTeX生成**:

```
プロンプト: この本のBibTeXエントリを作成してください:
著者名, 書籍タイトル, 出版社, 2023.
```

**結果**: 自動的にフィールドを埋めた完全なBibTeXエントリ

#### PDF → LaTeX変換（GPT-4以上）

```
プロンプト: 添付されたドキュメントのLaTeXコードを作成してください
```

**追加指示**: フォント、レイアウト、特定の要素を調整

## プロンプト設計のベストプラクティス

### プロンプト構築要素

- **役割定義**: 「あなたは...として振る舞う」「...のように行動する」
- **受信者の役割**: 子供、学生、教授など
- **欲しいもの**: 「私は...が欲しい」「私は...が必要」
- **タスク概要**: 「あなたは...すべき」「以下を実行してください」
- **除外事項**: 「...を使わない」（最初または再試行時）
- **トーン指定**: 「あなたのトーンは非公式で意欲的」
- **出力形式**: 箇条書き、番号付きリスト、表、特定言語のコード
- **コンテキスト提供**: 関連する背景情報すべて
- **例の提供**: 期待する出力の例

### トークン節約

- AIは人間ではないので「please」などの礼儀表現は省略可能
- プロンプトとテキストを合わせると包括的になるため、簡潔さが重要
- **プロンプトの再利用**: 同じプロンプトを複数のテキストに適用できる
  - 「このプロンプトをすべての後続入力に適用してください」と指示

### プロンプトの保存

- 有効なプロンプトを見つけたら、別の場所に保存
- 長期プロジェクトで再利用可能

## AI統合ツール

### editGPT (ブラウザ拡張機能)

**URL**: https://editgpt.app

**機能**:

- ChatGPTウェブサイトに統合される編集モード
- 削除部分を赤、追加部分を緑でハイライト
- 統計表示
- ボタン: 文法修正、校正、不自然な部分修正、フレーズ洗練

**インストール**:

1. Chrome/Firefox拡張機能ストアからインストール（Edge互換）
2. ChatGPTウェブサイトにアクセス
3. プロンプト入力後、**editGPT**ボタンをクリック
4. **Open in Editor**で専用編集ウィンドウを開く

**URL**: https://folivora.ai

**機能**:

- キーボードショートカット、マウスジェスチャー、ChatGPT統合

- 複数の設定可能

- 学術論文で訓練された言語ツール

## AI利用のベストプラクティス

### 著作権と盗用

**注意**:

- **AIの出力をそのままコピー&ペーストして使用することは避ける**
- AIの訓練データには他の著者の原文が含まれる可能性がある
- マシン処理されていても、元の表現が含まれる場合がある

**推奨**:

- AIの回答を参考に、自分の言葉で書く
- 創造性と正確性は著者の責任

### 正確性の検証

**AIの限界**:

- 論理的に見えても誤った情報を生成することがある
- 「ハルシネーション」（事実に基づかない情報）が発生する可能性

**対策**:

- **すべてのコードとコンテンツを検証**
- LaTeXコードはコンパイルして確認
- 事実情報は信頼できる情報源で再確認

### 新しいチャットの開始

**コンテキストのリセット**:

- 新しい主題には新しいチャットを開始
- 無関係な過去の会話がコンテキストに影響しないようにする

## まとめ

### AIの利点

- **迅速な回答**: 膨大な知識ベースから即座に情報を取得
- **コード生成**: LaTeXスニペット、テンプレート、複雑な構造を自動生成
- **コンテンツ改善**: 校正、言い換え、トーン調整、翻訳
- **ライターズブロック克服**: アイデア生成、アウトライン作成

### 注意点

- 機密データを入力しない
- 生成されたコードと情報を検証
- 盗用を避けるため、自分の言葉で書く
- 最新情報は公式ドキュメントで確認
