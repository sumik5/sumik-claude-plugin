## 1. 実験の二種類と目的

大学の実験には性質の異なる二種類があり、それぞれでレポートの書き方が変わる。

| 種類 | 特徴 | レポートの特徴 |
|------|------|--------------|
| **技能習得実験** | 低学年・短期間（2〜3コマ）・内容が決まっている | 「考察」のみに学生の主張が入る |
| **未知探究実験** | 高学年・長期間（1カ月〜1学期）・問題設定に自由度あり | 研究の意義を全体を通じて主張する |

**重要な違い**: 技能習得実験では「考察」が最も評価される。未知探究実験では「結果」と研究意義の主張が中心になる。

## 2. 実験レポートの基本構成

### 標準的な構成

```
表紙
  └── 授業名・実験課題名・所属学科名・氏名・学生番号・提出年月日

要旨

1. はじめに（目的）

2. 実験原理（技能習得実験では追加することが多い）

3. 実験方法

4. 結果

5. 考察（または「考察と結論」）

参考文献リスト

（感想）                        ← 任意、指定があれば記入
```

**注意**:
- 必ずページ番号を入れる
- 節番号は「1. はじめに」〜「5. 考察」に付ける
- 「要旨」「参考文献リスト」「感想」には節番号を付けない

### 卒業論文の場合

実験レポートに加え、以下が変わる：

| 項目 | 実験レポート | 卒業論文 |
|------|------------|---------|
| **はじめに** | 目的とその概要でよい | 研究意義の主張＋先行研究の紹介が必要 |
| **考察** | 結果の解釈程度でよい | 学術的な位置づけ・研究の意義まで論じる |
| **引用文献** | 参考文献（引用不要）が多い | 引用文献（本文内で引用）が基本 |
| **表紙** | 簡易 | 題目・所属・氏名・提出年月日（独立ページ） |

## 3. 要旨の書き方

### 要旨の役割

要旨は「**本文を読まずに理解できる小宇宙**」として機能する。

### 要旨に書く内容

| 要素 | 必要度 | 内容 |
|------|--------|------|
| ①実験の目的 | 必要な場合 | 何を明らかにしたいか |
| ②行った内容 | 必須 | どんな実験・測定を行ったか |
| ③得られた結果 | 必須 | 主な結果（具体的な数値を含める） |
| ④結果の解釈や意義 | 必要な場合（1文程度）| 結果が何を意味するか |

### 要旨の悪い例と良い例

**悪い例（抽象的すぎる）:**
```
本実験では気温の測定を行い、有意義な結果が得られた。
放射冷却について考察した。
```

**良い例（具体的・完結している）:**
```
夜間の気温低下に何が寄与するかを調べるために、気温・湿度・風速の観測を
2023年11月から12月にかけて行った。気温と天候との関係では、放射冷却に
期待されるとおり、晴天時に温度低下が大きく（平均3.2℃）、曇天時に温度
低下が小さかった（平均0.8℃）。また、平均風速が弱い場合（<2 m/s）に
温度低下が顕著であった。この関係は、風が強いと上空と地表付近の大気が
かき混ぜられるためと考えられる。
```

## 4. はじめに（目的・背景）の書き方

### 技能習得実験の場合

短いレポートでは背景を省略し、**目的のみ**書いてもよい。

```
本実験の目的は、XX法を用いてYYの測定を行い、ZZの関係を確認することである。
```

### 未知探究実験・卒業論文の場合

三段階（ホップ・ステップ・ジャンプ）で構成する。

| 段階 | 内容 | 方向性 |
|------|------|--------|
| **ホップ** | テーマが属する研究領域の重要性を主張 | 広い・一般的 |
| **ステップ** | 「しかし、〜が行われていない」と問題提起 | 中間 |
| **ジャンプ** | 「そこで本研究の目的は〜」と目的と手段を述べる | 狭い・具体的 |

**例:**
```
（ホップ）近年、XX現象の報告が局所的・大規模なスケールでなされている。
（ステップ）しかし、従来の報告のほとんどはXXについてのものであり、
YYについては報告がなされていなかった。
（ジャンプ）そこで本研究の目的は、YYの実態を明らかにすることである。
そのために、複数のデータセットを組み合わせてZZの変動パターンを同定する。
```

## 5. 実験原理の書き方

**この節の目的**: 自分が実験原理を理解していることを教員に示す。

### 書くべき内容

- 「はじめに」と「実験方法」をつなぐ橋渡し
- 測定することで何が明らかになるかを説明する
- 教科書の丸写しではなく、**自分が理解した内容を他人にわかるように書く**

### 注意点

- 関連する数式の意味を言葉でも説明する
- 専門用語は定義してから使う
- 長くなる場合は、本実験に直接関係する内容に絞る

---

## 6. 実験方法の書き方

### 記述の目標

**読者が実験を再現できる程度の詳しさ**で書く。

想定読者は「実験一般の知識はあるが、この特定の実験の知識はなく、実験教科書も持っていない人」。

### 必ず含めるべき情報

| 情報 | 例 |
|------|-----|
| 使用した試薬・材料の名前と量 | 「純水100 mL」「塩化ナトリウム（NaCl）5.0 g」 |
| 使用した装置・機器 | 「電子天秤（XX社製、型番YY）」 |
| 手順（どの方法を選んだかを明記）| 「沸騰したお湯に入れて5分間加熱した」（「5分間加熱した」だけではダメ） |
| 測定条件・パラメータ | 「温度：25℃、pH：7.0」 |

### 実験方法のタイトルは内容に合わせる

「実験方法」が適切でない場合は変更する：

| 内容 | 推奨タイトル |
|------|------------|
| 観測が主 | 「観測方法」 |
| データ解析が主 | 「データと解析手法」 |
| 数値計算が主 | 「数値計算モデル」 |

### 図の活用

装置図は**手描きで十分**。コンピュータの作図ソフトは時間がかかるため、手描きを推奨。

## 7. 結果の書き方

### 結果節の役割

「自分の結論を支える材料を示す部分」。未知探究実験・卒業論文では**最も重要な節**である。

### 「事実の記述」を守りながら意味を伝える

結果節では可能な限り**事実の記述スタイル**を維持する。

| Before（避けるべき）| After（推奨）|
|--------------------|-----------|
| 「放射冷却が気温変化に大きな役割を果たしていると、私は考える」（判断が入っている）| 「この結果は、放射冷却の特徴とよく一致している」（事実から離れずに意味を伝える） |
| 「図1を見ると、7月1日の最高気温が35.0℃であることがわかる」 | 「図1は、7月1日の最高気温が35.0℃であることを示している」または「7月1日の最高気温は35.0℃である（図1）」 |

### 図表の使い方

- 「結果」節の中心は図や表である
- 図にはその下に**図の説明文（キャプション）**をつける
- 表にはその上に**表の説明文**をつける
- 図表はそれ自身と説明文で内容が理解できなければならない
- 本文では図表の**特徴**を述べる。「図1は〜の図である」だけでは不十分

**よくある間違い**: 「結果」で図や表を出すだけで説明せず、「考察」で説明する
→ 図が示す事実は必ず「結果」節で述べること

## 8. 考察の書き方

### 考察の内容

技能習得実験では「考察」が最も評価される。考察に入れるべき内容：

1. **結果の要約**（考察の最初）: 全体の目的をふまえた結果のまとめ
2. **予測との比較**: 結果が予測と異なる場合に、その理由を分析
3. **推測・予想**: 結果から推測・予想される事項の説明
4. **問題点と改善提案**: 実験の問題点の議論および改善方法の提案

### 考察の最後には結論を書く

- 結論は「はじめに」で提起した目的に**対応している**こと
- 結論の記述が長くなる場合は、「結論」という節を別途設ける

### 考察の深め方

浅い考察：
```
気温は晴天時に低下した。これは放射冷却が起きたからだと思う。
```

深い考察：
```
気温は晴天時に大きく低下した（平均3.2℃）。この結果は放射冷却の
特徴と一致している。放射冷却とは、地表からの熱放射がさえぎられ
ない晴天時に冷却が大きくなる現象であり、本実験の結果はこれを支
持している。一方、同様の現象を報告した○○（参考文献1）では
平均2.8℃の低下が報告されており、本実験の結果は若干大きい。
これは、測定地点の標高（XX m）が高いため、大気中の水蒸気量が
少なく、放射冷却の効果が大きいためと考えられる。
```

## 9. 考察と感想の違い

| 項目 | 考察 | 感想 |
|------|------|------|
| **定義** | 結果の科学的解釈・分析・改善提案 | 個人的な印象・反応 |
| **例** | 「○○の作業で誤差が大きくなった原因は△△であり、□□という改善方法が考えられる」 | 「○○の作業はとても難しかった」「実験が長かった」 |
| **評価** | 論文の核心部分として評価される | 教員にとっては改善の参考になるが、本来の論文の評価対象外 |

**明確にすべきこと**: 問題点を具体的な改善方法まで提案できれば、それは立派な「考察」になる。

## 10. グラフ・図表の効果的な使い方

### グラフの種類と使い分け

| グラフの種類 | 適した用途 |
|------------|---------|
| **折れ線グラフ** | 時間経過や連続的な変化 |
| **棒グラフ** | カテゴリ間の比較 |
| **散布図** | 2変数の相関関係 |
| **ヒストグラム** | 度数分布 |
| **円グラフ** | 構成比率（ただし理系論文では少ない） |

### 良いグラフの条件

- **軸ラベルに単位を明記**:「時間 [s]」「温度 [℃]」
- **凡例が明確**
- **データ点にエラーバー**（標準偏差・標準誤差）を付ける
- **見やすいサイズ**のフォント（本文と同程度以上）

### 実験装置図

- 装置の全体像がわかる
- 重要な部品・接続関係が示されている
- 手描きで十分（ただし丁寧に）

## 11. よくある間違いと修正例

### 間違い1: 「結果」節での主観的判断

```
【悪い例】
夜間の気温は、晴天時に大きく低下した。したがって、放射冷却が気温変化に
大きな役割を果たしていると、私は考える。

【良い例】
夜間の気温は、晴天時に大きく低下した。この結果は、放射冷却の特徴とよく
一致している。
```

### 間違い2: 「図を見るとわかる」表現

```
【悪い例】
図1を見ると、7月1日の最高気温が35.0℃であることがわかる。

【良い例】
7月1日の最高気温は35.0℃である（図1）。
```

### 間違い3: 「実験方法」節での曖昧な記述

```
【悪い例】
試料を5分間加熱した。

【良い例】
試料を沸騰した水浴（100℃）中に入れ、5分間加熱した。
```

### 間違い4: 「考察」節での感想の混入

```
【悪い例】
この実験はとても難しかった。しかし、放射冷却の仕組みが理解できた。

【良い例】
（感想は別の「感想」節に書く）
考察には、「結果の解釈」「先行研究との比較」「実験の問題点と改善案」
「結論」を書く。
```

### 間違い5: 「要旨」での抽象的な記述

```
【悪い例】
この実験では有意義な結果が得られた。

【良い例】
△△の測定を行った結果、XX法では収率YY%、ZZ法では収率WW%が得られた。
XX法がZZ法より高い収率を示した原因は〜と考えられる。
```

## 12. 実験レポートのテンプレート

### 表紙

```
授業名：〇〇学実験
実験課題名：〇〇の測定
所属：〇〇学科
学生番号：123456
氏名：山田太郎
提出日：20XX年X月X日
```

### 要旨（100〜200字程度）

```
〇〇を明らかにするために、XXとYYを測定した。測定の結果、[具体的な結果]
が得られた。この結果は[先行研究・理論との比較]と一致しており、[意義・
結論]が示唆された。
```

### 1. はじめに（目的）

```
本実験の目的は〇〇である。（技能習得実験の場合）

【未知探究実験の場合】
（ホップ：背景・重要性）
（ステップ：問題提起）
（ジャンプ：本実験の目的と概要）
```

### 2. 実験原理

```
〇〇の原理を述べる。使用する数式：〇〇（数式の意味を言葉でも説明する）
```

### 3. 実験方法

```
3.1 使用試薬・材料
3.2 使用装置
3.3 実験手順
```

### 4. 結果

```
図・表を使って、得られたデータを提示する。
図表の特徴を本文で述べる。
```

### 5. 考察（と結論）

```
5.1 結果の要約
5.2 結果の解釈
5.3 先行研究との比較
5.4 実験の問題点と改善提案
5.5 結論（「はじめに」の目的に対応した結論を書く）
```

### 参考文献リスト

```
[1] 著者名（出版年）書名、出版社（書籍の場合）
[2] 著者名（出版年）論文名、雑誌名、巻、pp.XXX-YYY（論文の場合）
```
