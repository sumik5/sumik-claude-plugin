## 1. 三原則：明快・明確・簡潔

木下是雄（『レポートの組み立て方』）はレポートの文章に3つの原則を要求している。

| 原則 | 定義 | 妨げる要因 |
|------|------|-----------|
| **明快（Clarity）** | 論理の流れが自然で、一文と次の文の関係が即座にわかる | 論理順序の乱れ、文のつなぎ方の混乱 |
| **明確（Preciseness）** | 二通り以上に読めるところがなく、ずばりと断定的な表現 | ぼかし表現、曖昧な語句 |
| **簡潔（Conciseness）** | 一語を削れば必要な情報が不足するほど切りつめた表現 | 埋め草、冗長な言い回し |

### 簡潔さの基準（チャーチルの原則）

木下是雄はウィンストン・チャーチルの言葉を引用して、簡潔な文章の指針を示している。

> 「報告書をもっと短くするようにご配意ねがいたい。次のような言い方はやめよう：『次の諸点を心に留めておくことも重要である』、『……を実行する可能性も考慮すべきである』。この種のもってまわった言いまわしは埋め草にすぎない。省くか、一語で言い切れ。」

**Before（冗長）**:
```
学術研究は、真理の探究を目指す普遍的な知的活動であり、その成果は
広く人類共通の資産として蓄積されるものである。また、研究の発展は、
異なる経験や思考法を持つ人々の間の交流によって触発されることが多い
のは、歴史の教えるところである。
```

**After（簡潔）**:
```
学術研究の発展は、異なる経験や思考方法をもつ人々の間の交流によって
触発されることが多い。したがって、国際的な交流・協力は、研究活動を
進める上で欠くべからざる要素である。
```

## 2. 事実と意見の書き分け

レポートにおいて最も重要なルール。事実と意見の区別は「レポートと感想文を分かつ一線」である。

### 事実と意見の定義

| 種類 | 定義 | 例 |
|------|------|-----|
| **事実の記述** | しかるべきテストや調査によって真偽を客観的に判定できる記述 | 「東京の8月の平均気温は過去120年間おおむね上昇している」 |
| **意見** | ある前提にもとづく推論、判断、考え | 「この政策は効果がないと考える」 |
| **仮説** | 真偽は不明だが仮に打ち出した考え | 「気温上昇は都市化が原因ではないかと推測される」 |

### 事実の書き方のルール

```
○ 事実をそのまま断定形で書く
× 事実を推測のように書く（過小表現）
× 推測・解釈を事実のように書く（過大表現）
```

| 状況 | 推奨表現 | 避けるべき表現 |
|------|---------|-------------|
| 確認済みの事実 | 「〜である」「〜した」 | 「〜と思われる」「〜であろう」 |
| 可能性・推測 | 「〜かもしれない」「〜を示唆している」「〜であろう」 | 断定形（「〜である」） |
| 他者の意見 | 「○○によれば、〜」「○○は〜と述べている」 | 自分の意見のように書く |

**具体例**:

| 種類 | 例文 |
|------|------|
| ○ 事実を断定形で | 「東京の8月の気温は、過去120年間おおむね上昇している」 |
| × 事実を推測形で（誤り） | 「東京の8月の気温は、過去120年間おおむね上昇していると考えられる」 |
| × 推測を断定形で（誤り） | 「気温上昇は地球温暖化と関係している」 |
| ○ 推測を推測形で | 「気温上昇は地球温暖化あるいは都市化と関係しているかもしれない」 |

### 意見の書き方のルール

自分の意見を書くときの基本形は「私は、……と考える（推論する、思う、判断する）」。

```
○ 私は、この火事の原因は漏電であったと考える。（根拠がある推論）
○ この火事の原因は漏電であった。（証拠をあげて裏づけできる場合のみ）
× この火事の原因は漏電であったと思われる。（自分の責任が不明確）
```

他人の意見を引用する場合は、必ず「誰の意見か」を明示する。

```
○ ○○によれば、「引用文」とある。（出典明示）
× 「引用文」とある。（出典不明）
```

## 3. 一文一義と文長の原則

### 一文一義の原則

一つの文には一つの主張・事実のみを書く。複数の内容を一文に詰め込むと、主語・述語の対応が崩れ、読みにくくなる。

**Before（複数の内容が混在）**:
```
近年、情報化社会が進展し、インターネットの普及によって情報共有が
容易になり、その結果、学術情報へのアクセスが民主化されたが、
一方で情報の信頼性評価が重要な課題となっており、これに対応するため
情報リテラシー教育が注目されている。
```

**After（一文一義に分割）**:
```
近年、インターネットの普及により学術情報へのアクセスが容易になった。
その一方で、情報の信頼性評価が重要な課題として浮上している。
こうした状況を背景に、情報リテラシー教育への注目が高まっている。
```

### 適切な文長

| 場面 | 目安 |
|------|------|
| 一般的な文長 | 50〜80字（読点なしで長くなる場合は分割） |
| 上限 | 100字を超えたら分割を検討 |
| 複雑な説明 | 短文を恐れず積み上げる |

木下是雄は「すらすら読める文」の条件として「文は短く」を挙げ、接続詞「そして」「しかし」などを使って短文をつなぐことを推奨している。

## 4. 主語と述語の対応

主語と述語が対応していない文は、意味の取れない「悪文」の典型。

**悪例（主語・述語のねじれ）**:
```
留学生の学習目的が専門的で高度の専門用語を使用するような場合には、
その分野に必要な日本語が留学生に課せられることになる。
```

問題点：「学習目的が……課せられることになる」という主語・述語の対応がおかしい。

**改善例**:
```
留学生の学習分野が専門的で、高度に専門的な用語が多く使われる場合には、
彼らは専門用語と、その分野に特有な日本語の言い回しとを習得しなければ
ならない。
```

### 主語を省略しすぎない

日本語は主語を省略しやすいが、学術文章では「誰が行ったか」「誰の意見か」を明確にすることが重要。特にレビュー型レポートでは、先行研究の内容を卒論著者自身が行ったかのように書いてしまうミスが起きやすい。

```
× 「次にAとBとの関係を導出しよう」（卒論著者が導出したように読める）
○ 「彼らは次のようにAとBとの関係を導出した」（主語を明記）
```

## 5. 修飾語の配置ルール

修飾語は被修飾語の近くに置く。修飾語と被修飾語が離れると、どこにかかるかが不明確になる。

### 逆茂木型の文を避ける

木下是雄が「逆茂木型」と呼ぶ悪文パターン：修飾語が長く連なって、核心の語句が文末に来るまで意味が取れない文。

**悪例（逆茂木型）**:
```
昨年の夏から秋にかけて実施した、数十の農家を対象とした、
アンケートと聞き取り調査による、高齢農業従事者の労働実態調査の
結果を以下に報告する。
```

**改善例（修飾を整理）**:
```
以下では、高齢農業従事者の労働実態調査の結果を報告する。
この調査は昨年夏から秋にかけて実施し、数十の農家を対象に
アンケートと聞き取り調査を用いた。
```

### 修飾の順序の原則

- 長い修飾語は短い修飾語より前に置く
- 複数の修飾語は大きな範囲から小さな範囲へ
- 主節の動詞から遠い修飾ほど前に置く

## 6. 受動態と能動態の使い分け

### 能動態を基本とする

木下是雄は「文は能動態で書く」ことを原則として挙げている。受動態は文章をぼかし、責任の所在を曖昧にする。

| 状況 | 推奨 | 理由 |
|------|------|------|
| 通常の記述 | 能動態 | 主体が明確、文が簡潔 |
| 行為者が不明・不要な場合 | 受動態が可 | 例：「実験は2023年に実施された」 |
| 慣用的な受動表現 | 受動態が可 | 例：「〜と考えられている」「〜とされている」 |

**Before（受動態多用）**:
```
この調査は昨年3月に実施され、100名の学生によって回答が行われた。
その結果、以下の傾向が確認された。
```

**After（能動態に変換）**:
```
本研究は昨年3月に100名の学生を対象に調査を実施した。
その結果、以下の傾向が確認された。（確認された＝慣用的受動態として可）
```

## 7. 文体：「である」調と「です・ます」調

### 原則：「である」調を使う

学術的なレポート・論文では原則として**「である」調（常体）**を使う。

| 文体 | 特徴 | 使用場面 |
|------|------|---------|
| 「である」調（常体） | 簡潔・断定的・客観的 | レポート・論文・報告書 |
| 「です・ます」調（敬体） | 柔らかい・対話的・丁寧 | 手紙・スピーチ・プレゼン |

同一文書内で2つの文体を混在させてはならない。

### 断定形を恐れない

日本人はぼかした表現を好む傾向があるが（ドナルド・キーンの指摘）、学術文章では自分の考えを明確に言い切ることが必要。

| 避けるべき表現 | 推奨表現 |
|-------------|---------|
| 「……と言えないこともないかと思われる」 | 「……である」または「私は……と考える」 |
| 「……ではないかと思われる」 | 「……であると考えられる」（確実度が低い場合） |
| 「……について検討する必要があると思います」 | 「……を検討する必要がある」 |
| 「……ぐらい」「……たぶん」（不必要なぼかし） | ぼかし表現を削除するか、根拠を明示 |

## 8. 学術文章で避けるべき表現

### 口語・くだけた表現

| 避けるべき | 推奨 |
|----------|------|
| 「〜してしまう」 | 「〜する」「〜することになる」 |
| 「〜じゃない」「〜でしょ」 | 「〜ではない」「〜であろう」 |
| 「〜なんか」「〜とか」 | 「〜など」「〜や」「〜または」 |
| 「やっぱり」「やはり」（文頭） | 「これに対し」「しかし」等に変換 |
| 「すごく」「とても」（強調） | 「非常に」「著しく」または削除 |

### 曖昧・冗長な表現

| 避けるべき | 推奨 | 理由 |
|----------|------|------|
| 「〜という形で」「〜という風に」 | 削除または具体的表現に変換 | 「形」「風」が何も情報を加えない |
| 「〜のほうが」 | 「〜が」または比較の明示 | 不必要な回りくどさ |
| 「……についての研究」（タイトル） | 「……の研究」（「の」で十分） | 「について」が冗長 |
| 「〜の研究」（論文タイトル） | タイトルから「の研究」を削除 | 論文であることは自明 |
| 「〜ということがわかった」 | 「〜が明らかになった」「〜が確認された」 | 「ということ」が不要 |

### 感情・主観的表現

| 避けるべき | 理由 |
|----------|------|
| 「素晴らしい」「重要な」（根拠なし） | 評価の根拠が示されていない主観 |
| 「悲しいことに」「残念ながら」 | 筆者の感情混入 |
| 「言うまでもなく」「もちろん」 | 読者への押しつけ（ひとりよがり） |
| 「私たちは〜すべきだ」（断言的主張） | 根拠なしの主張。根拠を示してから主張する |

### 「もちろん」「当然」の罠

読者にとって当然でないことを「もちろん」「当然」で済ませる文章は、論証を省略している。

```
× 「もちろん、専門用語の学習は専門別におこなうべきだ」
○ 「専門用語の学習を一般日本語教育に組み込むことは困難である。
   なぜなら、専門分野ごとに語彙体系が大きく異なるためだ」
```

## 9. 句読点のルール

### 読点（、）の基本ルール

| 使用場面 | 例 |
|---------|-----|
| 主語の後（長い主語） | 「この実験の結果は、以下の通りである。」 |
| 逆接の接続詞の後 | 「しかし、データは異なる傾向を示した。」 |
| 長い修飾語の後 | 「詳細な分析を経て、次の結論に達した。」 |
| 列挙（最後の項目以外） | 「A、B、およびCから構成される。」 |
| 読みやすさのための区切り | 意味のまとまりに応じて適宜使用 |

### 句点（。）のルール

- 文の終わりには必ず句点を打つ
- 箇条書き・見出しには句点を打たないことが一般的
- カッコ内の文に句点は打たない（例：「（詳細は後述する）」）

### 横書きの句読点

現代の横書き学術文章では次の2種類が使われる。

| 組み合わせ | 使用場面 |
|----------|---------|
| 「、」と「。」 | 一般的な日本語横書き文書 |
| 「，」と「．」（半角） | 理系学術論文、国際会議論文、一部の学会誌 |

指定がある場合はそれに従う。一つの文書内で混在させない。

### 「…」（三点リーダ）と「——」（ダッシュ）

学術文章では原則として使用を避ける。引用文内に省略がある場合に「……」を使うのは可。
