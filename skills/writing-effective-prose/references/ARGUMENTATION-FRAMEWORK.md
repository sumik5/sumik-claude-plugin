## 1. 論文の本質：「問い・主張・論証」の三点セット

論文とは、問いに対して明確な答えを主張し、その主張を論理的に裏づけるための事実的・理論的な根拠を提示する文章である。

```
（１）与えられた問い、あるいは自分で立てた問いに対して、
（２）一つの明確な答えを主張し、
（３）その主張を論理的に裏づけるための事実的・理論的な根拠を提示して主張を論証する。
```

この三点が揃ってはじめて「論文」と呼べる。逆に言えば、どれか一つが欠けていれば論文ではない。

### 論文とそれ以外の違い

| 文章の種類 | 問い | 主張 | 論証 |
|-----------|------|------|------|
| **論文** | あり | 明確・一つ | 根拠付き |
| 読書感想文 | なし | 感想のみ | なし |
| 新聞コラム | あいまい | 連想・飛躍で展開 | 弱い |
| エッセイ | あいまい | 印象・体験 | なし |
| 報告書（事実のみ） | なし | なし | なし |

## 2. 論証の基本形式

論証とは「自分の答えを論理的にサポートする証拠を効果的に配列したもの」である。

### 2-1. 演繹的論証（妥当性が最も高い）

前提が真であれば結論が必ず真になる。

```
【形式】
前提１：すべてのAはBである
前提２：CはAである
結論：CはBである

【例】
前提１：道徳的に考慮されるべき存在は苦痛を感じることができる
前提２：動物は苦痛を感じることができる
結論：動物は道徳的に考慮されるべき存在である
```

**注意点**: 前提の真偽が問われる。前提が疑わしければ、その前提自体も論証が必要。

### 2-2. 帰納的論証（ちょっと弱い）

個別事例から一般原則を導く。反例が一つでもあれば崩れる。

```
【形式】
事例A、B、C、D……においてXが観察された
→ 一般的にXが成り立つ（可能性が高い）

【例】
調査したすべての事例で、企業倫理教育を受けた技術者は
内部告発率が高かった
→ 企業倫理教育は内部告発を促す効果がある（可能性が高い）
```

**注意点**: サンプルの偏りに注意。「確実に」ではなく「おそらく」という言い方が誠実。

### 2-3. アブダクション（仮説形成）

最もよい説明を選ぶ推論。科学的仮説の形成によく使われる。

```
【形式】
現象Xが観察された
仮説Hが真であれば、Xはうまく説明できる
Hに匹敵するよい説明は他にない
→ おそらくHが真である

【例】
チャレンジャー号は直前まで打ち上げ延期を主張していたのに
なぜ急に態度を変えたか？
「集団思考」仮説で最もうまく説明できる
→ サ社の態度変更は集団思考によるものだった（可能性が高い）
```

### 論証形式の比較

| 形式 | 論理的強さ | 反証可能性 | 使用場面 |
|------|----------|-----------|---------|
| 演繹 | 最も強い | 前提次第 | 理論的結論の導出 |
| 帰納 | 中程度 | 反例で崩れる | データから傾向を示す |
| アブダクション | 相対的 | 別の説明が登場すれば崩れる | 仮説の提示・現象の説明 |

## 3. 根拠の種類と強さ

論証に使える根拠には種類がある。適切な根拠を選択することが論証の質を決める。

| 根拠の種類 | 説明 | 強さ | 注意点 |
|-----------|------|------|--------|
| **定量的データ** | 統計、調査結果、実験データ | 強い | 出典・調査方法の明示が必須 |
| **先行研究** | 学術論文・専門書の知見 | 強い | 引用・参照の正確な明示が必須 |
| **論理的推論** | 演繹・帰納による推論 | 中〜強い | 前提の妥当性を示す必要あり |
| **事例・具体例** | 具体的な出来事・ケース | 弱〜中 | 代表性・典型性を示す必要あり |
| **権威への訴え** | 専門家の意見 | 弱い | 論証の補強には使えるが主な論拠にはならない |
| **感情・経験** | 個人の感情・体験 | ほぼ無効 | 論文の根拠としては使えない |

### 「原因」と「根拠」の混同（よくある失敗）

「自分がその意見をもつようになった経緯・感情的な動機」と「その主張が正しいことを示す根拠・理由」は別物。

| 種類 | 例 | 論拠になるか |
|------|-----|-------------|
| 原因（経緯） | 「動物実験の映像を見てショックを受けた」 | なし |
| 根拠（理由） | 「行われている動物実験の80%は代替手段が存在する（○○調査）」 | あり |

## 4. 反論への対処法

優れた論文は、想定される反論を先回りして処理する。反論を無視すると論証が片手落ちになる。

### 反論対処の基本パターン

```
【譲歩+反論+再主張】

確かに（相手の主張を認める部分）、
一定の妥当性はある。しかし（反論）、
○○という理由から（根拠）、
やはり（自分の主張）が正しいと言える。
```

### 反論対処の具体例

**Before（反論を無視した弱い論証）**:
> 動物には権利を認めるべきである。なぜなら動物も苦痛を感じるからだ。

**After（反論を処理した強い論証）**:
> 動物には基本的な権利を認めるべきである。なぜなら動物も苦痛を感じる能力をもち、その苦痛を回避する利益は道徳的考慮に値するからだ。「動物に権利を認めると人間の食料・医療が脅かされる」という反論もあるが、権利の認定は「一切の利用禁止」を意味しない。苦痛を最小化する義務と両立する形での利用は引き続き認められる。

### 反論の強度と対処策

| 反論の強度 | 対処策 |
|-----------|--------|
| 弱い反論（前提が誤り） | 前提の誤りを指摘して退ける |
| 中程度の反論（部分的に妥当） | 「確かに〜だが」と認めつつ自説の優位性を示す |
| 強い反論（見逃せない問題点） | 主張の範囲を限定するか、反論を踏まえて主張を修正する |

## 5. 論理の飛躍を避ける方法

### 5-1. 論証の連鎖を丁寧につなぐ

大きな主張は、いくつかの小さな問いに分解して順番に論証する。

```
大きな主張：「動物実験は倫理的に許されない」

分解：
  問い１：動物は苦痛を感じることができるか → 根拠を示す
  問い２：苦痛を感じる存在の利益は道徳的考慮に値するか → 根拠を示す
  問い３：動物実験は不必要な苦痛を生じさせているか → データを示す
  問い４：代替手段は存在するか → 根拠を示す
  結論：以上より、動物実験は倫理的に許されない
```

### 5-2. 各論証ステップを確認する問い

各主張に対して自問する:
- 「なぜそう言えるか？（Why so?）」— 根拠は何か
- 「それだけで十分か？」— 別の説明が可能ではないか
- 「例外はないか？」— 反例は想定できないか

### 5-3. 「要するにどういうことか（So what?）」を問い続ける

データや事実を並べるだけでは論証にならない。「これらの事実が意味することは何か」を必ず引き出す。

| パターン | 問題 |
|---------|------|
| 「以下のとおりである」と箇条書きを並べて終わる | So what? の放棄 |
| 「さまざまな要因がある」で終わる | 意味の引き出しを放棄 |
| 「一概には言えない」で終わる | 主張の放棄 |

## 6. ダメな論証パターン

### 6-1. 循環論法（論点先取）

結論をすでに前提に含めている。証明していないことを証明に使う。

```
❌ 「Aは正しい。なぜならAが正しいことは明らかだからだ」
❌ 「聖書は神の言葉である。なぜなら聖書にそう書いてあるからだ」
```

### 6-2. 相関と因果の混同

同時に起きた（または前後して起きた）ことを「原因・結果」と決めつける。

```
❌ 「アイスクリームの売り上げが上がると水難事故が増える
   → アイスクリームが水難事故を引き起こす」
（正しくは：どちらも気温上昇という共通原因による相関）

❌ 「AのあとにBが起きた → AがBを引き起こした」
（前後関係 ≠ 因果関係）
```

### 6-3. 権威への訴え（不当な権威論証）

専門家の意見を論拠にすることは補強にはなるが、主な論拠にはなりにくい。とくに専門外の権威や対立意見がある分野では。

```
❌ 「A教授がそう言っているので正しい」（反証なし）
✅ 「A教授の研究（○○、△△年）によると〜とされている。
   これと同様の結果は△△の研究でも確認されている（○○）」
```

### 6-4. 人身攻撃（Ad hominem）

主張の内容ではなく主張者の属性・性格を攻撃する。

```
❌ 「Aは専門家ではないから、この主張は信用できない」
❌ 「Aは会社から研究費をもらっているから、この結論は信用できない」
（利益相反の可能性を示すことはできるが、主張の内容はそれとは別）
```

### 6-5. 滑り坂論法（Slippery slope）

一つの変化が、論証なく連鎖的に悪結果をもたらすと決めつける。

```
❌ 「安楽死を認めたら、次は知的障害者の安楽死も認められ、
   やがてすべての弱者が抹殺される社会になる」
（各ステップに論証が必要）
```

### 6-6. わら人形論法（Straw man）

相手の主張を実際より弱く・極端に歪めて攻撃する。

```
❌ 相手：「動物実験に一定の制限を設けるべきだ」
   自分：「動物実験をすべて禁止するという意見は非現実的だ」
（相手はそんなことを言っていない）
```

## 7. 論争型論文と実証型論文

論文には大きく二つのタイプがある。

| 項目 | 論争型論文 | 実証型論文 |
|------|-----------|-----------|
| **目的** | 特定の主張・立場を論証する | 現象・事実の分析・記述 |
| **問いの型** | Should/Is it right to...? | What/Why/How? |
| **主な根拠** | 先行研究・論理的推論 | データ・観察・実験 |
| **典型的な構成** | 問題提起→主張→論証→反論処理 | 問題→方法→結果→考察 |
| **例** | 「安楽死は許されるか」 | 「なぜ企業不祥事は防げないのか」 |

## 8. 主張の強さの段階

論証の強さに応じて、適切な強さの言い方を選ぶ。過度に断定的な表現は証拠が伴わないと逆効果。

| 段階 | 表現例 | 使用場面 |
|------|--------|---------|
| **断定** | 「〜である」「〜と言える」 | 演繹的に結論が導かれる場合 |
| **強い推量** | 「〜と考えられる」「〜可能性が高い」 | 根拠が十分で反証が少ない場合 |
| **推量** | 「〜と思われる」「〜かもしれない」 | 根拠はあるが不確実性が高い場合 |
| **提案** | 「〜すべきである」「〜を提案する」 | 価値判断・政策的主張 |
| **限定断定** | 「少なくとも〜の場合は」「〜という条件下では」 | 例外や限界を明示した上での主張 |

### Before/After：主張の強さの調整

**Before（過度な断定）**:
> 動物実験はすべて倫理的に許されない。

**After（適切な限定）**:
> 現在行われている動物実験の大部分は、代替手段が存在するにもかかわらず行われており、その点で倫理的に正当化が難しい。ただし、代替手段が存在せず人命に直接関わる研究については、厳格な倫理審査を前提に一定の例外を認めることができる可能性がある。

## 9. チェックリスト

### 論証の基本構造
- [ ] 論文に明確な「問い」があるか
- [ ] 問いに対する「主張（答え）」が一つ明確に示されているか
- [ ] 主張を支える「根拠」が複数示されているか
- [ ] 根拠と主張のつながりに論理的飛躍がないか

### 根拠の質
- [ ] 根拠に出典（引用・参照）が明示されているか
- [ ] 「感情的動機」と「論拠」を混同していないか
- [ ] 相関と因果を混同していないか
- [ ] 主な根拠が権威への訴えだけになっていないか

### 反論の処理
- [ ] 想定される主な反論を挙げているか
- [ ] 反論に対して正面から応答しているか
- [ ] 反論を受けて主張の範囲を適切に限定しているか

### 論証パターンの確認
- [ ] 循環論法になっていないか
- [ ] わら人形論法を使っていないか
- [ ] So what? を放棄した「以下のとおり」で終わっていないか
- [ ] 主張の強さは根拠に見合っているか（過度な断定/過度な弱腰になっていないか）
