## 1. 引用の役割と種類

### なぜ引用するのか

アカデミックな世界の基本ルール：

> 「人がそれなりの努力を傾注して調べたり考えたりして到達した真理・知識は、人類すべてのものとして共有されるべきである。しかし、それを生みだした人にはそれ相当の尊敬が払われなければならない。」

引用のルールを守ることで、**自分のオリジナリティ**と**他者の業績**が明確に区別される。

### 引用の三種類

| 種類 | 定義 | 使い分け |
|------|------|---------|
| **直接引用（原文引用）** | 原文をそのまま転載する | 原文の表現そのものが重要な場合 |
| **間接引用（パラフレーズ）** | 内容を自分の言葉で書き換える | 内容が重要だが表現は重要でない場合 |
| **要約引用** | 複数の箇所や文献全体を要約して引用する | 全体的な主張・データの傾向を示す場合 |

### 引用の量的基準（目安）

| 条件 | 目安 |
|------|------|
| 1つの引用の長さ | 400字以内 |
| 全引用の総量 | 自分が書く全文の20%以内 |

これを超える場合は著作権者の許諾が必要。

## 2. 直接引用のルール

### 短い引用（2行以内）

カギ括弧「　」でくくり、本文に埋め込む。

```
○○は「自分のことから文章作法というような文を書くのは、大変むつかしい。
いったいどうやって自分は文章を書いているのだろう、と改めて考えてしまう」
と述べている（著者名 年, p.XX）。
```

### 長い引用（3行以上）

前後を1行空けて、本文より2〜5字分インデントして書く。カギ括弧は不要。

```
（本文の続き）......と述べている。

　この考えを踏まえると、文章作法を教えることの困難さは
　単に技法の問題ではなく、書き手の内省プロセスに深く
　関わるものと言えよう。（著者名 年, pp.XX-XX）

なお、この問題については......（本文に戻る）
```

### 引用の鉄則

- かなづかい・漢字の使い方・句読点を**原文どおり**に転載する
- 旧仮名遣い・旧字体を含む場合もそのまま引用し、変更した場合は「現代仮名遣いに改めた」と注記する

## 3. 間接引用（パラフレーズ）の書き方

### なぜパラフレーズするのか

- 原文が長すぎて直接引用に向かない
- 原文より自分の言葉の方が論旨に合う
- 内容（事実・主張）が重要で表現は重要でない

### 書き方

地の文として書き、出典を末尾に示す。

```
シンプソンは、発電所内の労働者は自分たちが危険な物質を扱っているという
意識が希薄だということを報告している（シンプソン 1980, pp. 156-8）。
```

### 要約引用

複数の文献をまとめて参照する場合：

```
気候レジーム・シフトは多くの注目を集めてきた（XXX 1999; YYY 2000; ZZZ 2001）。
```

## 4. 引用文への加工が許される範囲

直接引用では原文を忠実に転載するが、以下の加工は許されている。

### （1）省略：「（中略）」

読者に注目してほしい箇所が前後に分かれており、間を省く場合。

```
「認識論は、われわれは現にいかにして知識を獲得しているかという事実問題だけでなく、
われわれはいかにして知識を獲得すべきかという規範的問題も問わねばならない。（中略）
クワインの議論はあまりにも粗雑だと言わざるをえない」（著者 年, pp.XX-XX）。
```

### （2）傍点・強調：「（傍点引用者）」

引用者が強調を加えた場合は、原文には強調がないことを明示する。

```
クワインは、「だから認識論は自然科学内部の企てであり、[心理学]の一章なのである」
（Quine 1969, p. 65、傍点引用者）と結論している。
```

### （3）誤字・脱字：「（ママ）」

原文に誤字脱字がある場合、そのまま引用し（ママ）と注記する。

```
「進化の課程（ママ）で作られた人間の表情は......」（著者 年, p.XX）。
```

### （4）補足説明：「（括弧内引用者）」

引用文から切り出したことで指示語の指示先が不明になる場合などに補足。

```
クワインは、「だからそれ（＝認識論）は自然科学内部の企てであり......」
（Quine 1969, p. 65、括弧内引用者）と結論している。
```

### （5）段落の区切り：スラッシュ「／」

2つの段落にまたがる箇所を引用する場合、段落の区切りにスラッシュを使う。

## 5. 注の種類と使い分け

### 注を使う目的

本文中でくどくど説明すると、論旨の理解を妨げる情報を本文と離れた場所で説明する。

**注に入れる内容の例:**
- 特殊な用語の説明
- 出典・参照先の情報
- 人物・出来事の補足説明
- 本文から外れると話題が脱線するが触れておきたい情報

### 注の種類

| 種類 | 位置 | 特徴 |
|------|------|------|
| **脚注（footnote）** | 各ページの下部 | その場ですぐ確認できる |
| **後注（尾注、endnote）** | 本文の後ろにまとめて | ページ下が煩雑にならない |
| **文中注（括弧注）** | 本文中に括弧で挿入 | 著者名＋年で示す（後述） |

### 使い分けの指針

- **人文系・文学系**: 脚注・後注が多い
- **社会科学系・理系**: 文中注（著者・年方式）が標準的
- **提出先の教員・ジャーナルのルール**に従う（提出先の論文スタイルを真似るのが最善策）

## 6. 本文中での引用箇所の示し方

### 著者・年方式（文中注方式）

最も広く使われる形式。

```
（著者姓 出版年, p. ページ番号）
（著者姓 出版年, pp. 開始ページ-終了ページ）
```

**例:**

| 状況 | 表示 |
|------|------|
| 単著の1ページ | （黒沢 1998, p. 12）|
| 単著の複数ページ | （黒沢 1998, pp. 123-4）|
| 共著（著者2名）| （黒沢・谷崎 1999, p. 12）|
| 共著（著者多数）| （谷崎他 2000, p. 12）|
| 英語文献・単著 | （Simpson 1999, pp. 529-31）|
| 英語文献・2著者 | （Monroe & Simpson 1988, p. 52）|
| 英語文献・多著者 | （Monroe et al. 1988, p. 145）|
| 同一著者同年の複数著作 | （黒沢 1998a, p. 12）（黒沢 1998b, pp. 123-4）|

**注意**:
- 「pp.」の「p」は小文字、ピリオドを忘れない
- ページ数の「123-4」は「123-124」の略（繰り返し数字を省略する）

### 番号注方式

本文に肩数字（上付き番号）をつけ、対応する注を脚注・後注に記載する。

```
（本文）科学実在論は独立性テーゼと知識テーゼという二つのテーゼをともに
主張する立場だ（23）。

（注23）科学実在論のこの定式化は（Papineau 1996, p. 2）による。......
```

## 7. 参考文献リストの書式

### 書籍（単著）

```
【日本語】
著者姓名（出版年）『書名』出版社

【英語】
著者姓, イニシャル（出版年）, タイトル（イタリック or アンダーライン）, 出版社

【例（日本語）】
木下是雄（1981）『理科系の作文技術』中央公論社

【例（英語）】
Simpson, H. (1999), From Springfield: A Life and Opinion of a Nuclear Plant Worker, Oxbridge University Press
```

### 書籍（共著）

```
【日本語】
著者A・著者B（出版年）『書名』出版社

【例】
黒沢みなも・谷崎ゆかり（1999）『現代教育論』とちり書房
谷崎ゆかり他（2000）『不良教員を斬る』ぬるをブックス

【英語】
Monroe, M. & Simpson, M. (1988), Family Therapy, Tombstone Books
Monroe, M. et al. (1988), Rich People with Emotional Problems, Bedpan & Co.
```

### 翻訳書

```
著者名（訳者名訳）（翻訳出版年）『日本語タイトル』出版社

【例】
シンプソン・H（美浜ちよ訳）（1980）『スプリングフィールド便り──ある原発労働者の生活と意見』わかよ堂
```

### 雑誌論文

```
【日本語】
著者名（出版年）「論文タイトル」『雑誌名』巻（号）, pp. 開始-終了ページ

【英語】
著者姓, イニシャル（出版年）: 論文タイトル, 雑誌名（イタリック）, 巻, 開始-終了ページ, doi:XXXX（記載できる場合）

【例（日本語）】
見延庄士郎（2003）「長期変動とレジーム・シフト」『月刊海洋』35, pp. 87-94

【例（英語）】
Minobe, S., A. Kuwano-Yoshida, N. Komori, S.-P. Xie, and R. J. Small (2008): Influence of the Gulf Stream on the troposphere, Nature, 452, pp. 206-209, doi:10.1038/nature06690
```

### Webサイト

```
著者名（作成・更新年）「ページタイトル」ウェブサイト名, URL（アクセス日:YYYY年MM月DD日）

【例】
文部科学省（2023）「学術論文の引用ルールについて」文部科学省ウェブサイト,
https://www.mext.go.jp/... （アクセス日：2024年1月15日）
```

### リストの並び順

- 著者姓のアルファベット順（英語文献）または五十音順（日本語文献）
- 同一著者の複数著作は出版年順
- 同一著者・同年の場合は (1998a), (1998b) と区別

## 8. 日本語と英語文献の混在

### 混在時のルール

- 同一の参考文献リストに日本語文献と英語文献を混在させる場合は、**著者名の読み**でアルファベット順・五十音順に統合するか、日本語と英語に分けて二つのリストにする
- どちらの方法でも、**リスト内で書式を統一**する

### 英語文献の著者名の表記

| 状況 | 表示 |
|------|------|
| 参考文献リスト | 姓, イニシャル（フルネームでも可）|
| 本文中での言及 | 姓のみ（例：Simpson）|
| 本文中での言及（日本語文） | カタカナ表記が慣習的（例：シンプソン）|

## 9. 主な引用スタイルの種類

### SIST02（日本）

日本のST（科学技術情報流通技術基準）に基づく標準的な書式。多くの日本語学術誌で使用。

### APA（アメリカ心理学会）スタイル

心理学・教育学・社会科学系で広く使われる。著者・年方式が基本。

### MLA（現代語学文学協会）スタイル

人文学・文学・語学系で使われる。

### Chicago スタイル

歴史学・人文系で使われる。注番号方式と著者・年方式の両方がある。

### 学会・雑誌ごとのスタイル

最も確実なのは「提出先の雑誌や教員が書いた論文のスタイルを真似る」こと。

## 10. 文献管理のヒント

### 文献管理ツールの活用

| ツール | 特徴 |
|--------|------|
| **Zotero**（無料）| ブラウザ拡張機能でWeb上の文献を自動収集。多形式に対応 |
| **Mendeley**（無料）| PDF管理に強い。クラウド同期あり |
| **Endnote**（有料）| 機能が充実。研究機関での利用が多い |

### 文献収集時の注意

- 引用する可能性があるものはすべてメモしておく（著者名・年・書名・ページ数）
- 「あとで調べよう」はほぼ確実に忘れる
- コピー・要約・原文引用の区別がわかるようにシルシをつけておく

### 著作権の基本

- 公表された著作物は、**出典を明示**すれば引用・利用できる（著作権法32条）
- 図・写真・表を転載する場合は著作権者の**許可が必要**
- 私信や私的な対話で得た情報は、相手の許可なしに引用しない
