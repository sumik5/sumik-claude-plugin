## 1. 三部構成の原則

学術的なレポート・論文の基本構成は「序論・本論・結論」の三部からなる。それぞれの役割と分量配分の目安は以下の通り。

| パート | 別名 | 役割 | 分量目安 |
|--------|------|------|----------|
| **序論** | はじめに / イントロダクション | 読者を本論に誘導する「導入部」。問題設定・背景・調査方法の概説・結論の予告 | 全体の10〜15% |
| **本論** | 本文 / ボディ | 調査・研究の具体的な内容と結果。レポートの核心部分 | 全体の70〜80% |
| **結論** | おわりに / まとめ | 本論の結果を要約し、自分の主張を明確に示す。今後の課題・展望 | 全体の10〜15% |

### 三部構成の論理的な流れ

```
序論：「問題は何か。なぜ重要か。どう調査したか。何がわかったか（要旨）」
  ↓
本論：「調査の方法 → 調査の結果 → 事実の記述」
  ↓
結論：「結果のまとめ → 自分の主張 → 今後の課題」
```

## 2. 序論の書き方

序論は読者を本論にスムーズに誘導するための「準備段階」。以下の要素を含める。

### 序論に盛り込む内容

| 要素 | 説明 | 必要度 |
|------|------|--------|
| (a) 話題の提示 | このレポートで取り上げる問題・テーマは何か | 必須 |
| (b) 取り上げた動機・背景 | なぜその問題を選んだのか。問題の社会的・学術的背景 | 必須 |
| (c) 先行研究の概観 | 従来の研究ではどこまでわかっているか（必要な場合） | 状況による |
| (d) 調査・研究の方法 | どんな方法で調査・研究をおこなったか（2〜3行程度） | 必須 |
| (e) 結論の提示 | **どんな結論に至ったかを簡潔に示す** | 強く推奨 |

**重要**: (e) の結論の予告は省かれることが多いが、木下是雄（『レポートの組み立て方』）は「たとえ古典的な序論・本論・結論型でも、序論の末尾に結論を簡潔に示すことを強く勧める」と述べている。

### 重点先行の原則

現代の情報化社会では、最も重要なポイント（要点）を冒頭に書く「重点先行」が求められる。

**Before（重点後行・非推奨）**:
```
わが国では近年、高齢化が急速に進んでいる。これに伴い介護問題が
社会的な課題となってきた。さまざまな調査が行われており、本稿では
こうした状況を踏まえて検討する。
```

**After（重点先行・推奨）**:
```
本稿は、わが国の在宅介護における家族負担の実態を調査し、
公的支援制度の拡充が最優先課題であることを明らかにする。
```

## 3. 本論の書き方

本論はレポートの大半を占め、「調査の方法」と「調査の結果」の2部からなる。

### 本論の構成要素

| 要素 | 内容 | 注意点 |
|------|------|--------|
| 調査・研究の方法 | 実際にやったことを具体的に記述 | 読んだ人が追認できる程度の詳細さ |
| 調査・研究の結果 | 明らかになった事実の記述 | **事実の記述が中心**。主観混入を避ける |
| 結論（B型の場合） | 事実から導かれる主張 | 論理的過程を明記。あいまいさを排除 |

### 本論における事実と意見の割合

木下是雄が大学生に課したレポートでは「内容の2/3以上が事実の記述であること」を要求している。これが本論の基本的な構成比率の目安となる。

```
本論の理想的な構成比率
├── 事実の記述（事実・引用・データ）：65〜70%
└── 意見・解釈・考察：30〜35%
    ※意見は必ず「根拠を示した意見」であること
```

## 4. 結論の書き方

結論の役割は主に2つ。本論の結果を「まとめる」ことと、「自分の主張を組み立てる」ことである。

### A型（序論・本論・結論型）の結論

1. 本論の「調査・研究の結果」を簡明にまとめて列挙
2. それにもとづいて自分の主張・見解を組み立てる
3. 必要に応じて調査の意義・今後の課題を展望する

### B型（概要・序論・本論・論議型）の場合

「結論」は本論の末尾に入れ、独立の節を設けない。代わりに「論議（考察）」の節を設けて、自分の研究を第三者的に検討する。

### 結論で避けるべき表現

| 避けるべき | 改善例 |
|----------|-------|
| 「以上のことから、このような傾向があると思われます」 | 「以上の調査から、Xという傾向が明らかになった」 |
| 「まとめると、さまざまな問題があることがわかりました」 | 「本調査で明らかになった主要な問題は次の3点である」 |
| 「今後さらに研究が必要と思われます」 | 「今後の課題は○○の検討である」 |

## 5. パラグラフ・ライティングの原則

パラグラフ（段落）は説明・論述文の基本構成単位。欧米のレトリック教育では最も重要な要素として徹底的に教える。

### パラグラフの定義

> 「文章の一区切りで、内容的に連結されたいくつかの文から成り、全体として、ある一つの話題についてある一つのこと（考え）を言う（記述する、主張する）もの」
> — 木下是雄『レポートの組み立て方』

### 1パラグラフ＝1トピックの原則

| 構成要素 | 役割 | 配置 |
|---------|------|------|
| **トピックセンテンス（中心文）** | そのパラグラフで何を言うかを一文で述べる | 原則としてパラグラフの冒頭 |
| **サポーティングセンテンス（展開文）** | 中心文の内容を具体的に詳説・証明する | 中心文の後 |
| **コンクルーディングセンテンス（結び文）** | 必要に応じてパラグラフ全体を結ぶ | パラグラフの末尾（省略可） |

### パラグラフ構成の例

**悪い例（中心文がない）**:
```
日本では少子化が問題になっている。出生率が低下している。
女性の社会進出が進んでいる。結婚年齢も上がっている。
```

**良い例（中心文＋展開文）**:
```
日本の少子化は、複合的な要因が絡み合う構造的問題である。[中心文]
出生率は2023年に過去最低の1.20を記録した。[展開文・事実]
その背景には、女性の社会進出による晩婚化、育児と仕事の両立困難、
高い教育費への不安などが挙げられる。[展開文・説明]
```

### 中心文の位置

欧米流の作文教育では「中心文はパラグラフの冒頭に置け」と教える。これにより「各パラグラフの第1文だけをひろい読みすれば全体の概要がつかめる」文章になる。

日本語では述語が文末にくる構文的特性から、中心文を末尾に置くほうが書きやすい場合もある。ただし、**中心文のないパラグラフは避ける**こと。

## 6. 段落の接続と論理の流れ

### 接続表現の分類と使用例

| 論理関係 | 接続表現 | 使用例 |
|---------|---------|-------|
| **順接・付加** | したがって、そのため、その結果、また、さらに | 「以上の調査結果から、したがってXが必要である」 |
| **逆接・転換** | しかし、ところが、一方、これに対して | 「賛成意見が多い。しかし、反対意見も無視できない」 |
| **例示・具体化** | たとえば、具体的には、すなわち | 「問題がある。たとえば、〇〇という事例がある」 |
| **対比** | 一方、他方、これに対して | 「Aは〇〇である。一方、Bは△△である」 |
| **まとめ** | 以上のことから、このように、要するに | 「以上のことから、Xが有効な手段といえる」 |

### 「が」の多用に注意

清水幾太郎（『論文の書き方』岩波新書）は「が」を「小さい魔物」と呼び、多用を警告している。「〜だが、〜」という順接・逆接の両方に使える曖昧な接続は、論理の流れを不明確にする。

| 避けるべき | 改善例 |
|----------|-------|
| 「Aであるが、Bである」（関係不明） | 「Aである。しかし、Bである」（逆接を明示） |
| 「AだがBだがCだが」（連鎖） | 文を分割し、それぞれの接続を明示する |

## 7. 見出し・セクション分けのルール

### 見出しの機能

レポートのような情報伝達を主目的とする文書では、こまめに見出しをつけて「どこに何が書いてあるか」を見やすくすることが大切。

### 見出しの種類と記法

| レベル | 用途 | 記法例（理系） | 記法例（文系） |
|-------|------|-------------|-------------|
| 大見出し（章） | 序論・本論・結論など | 1. / 第1章 | Ⅰ. |
| 中見出し（節） | 各章の主要セクション | 1.1 / 1.2 | A. / B. |
| 小見出し（小節） | 節内のトピック | 1.1.1 | 1. / 2. |

### セクション分けの注意

- 「序論」「本論」「結論」という見出しをそのまま使うのは適当でない場合が多い
- **内容を表す具体的な見出し**をつけることが原則
- 同じレベルの見出しは並列関係にあること（上位・下位関係にしない）

**悪い例**:
```
1. 序論
2. 問題の背景    ← 「序論」の一部なのに同レベル
3. 本論
```

**良い例**:
```
1. 問題の背景と研究目的
2. 調査の方法と対象
3. 調査結果
4. 考察と結論
```

## 8. 構成のバリエーション

### A型（古典的構成）vs B型（現代的構成）

木下是雄は2種類の構成を整理している。

| 比較項目 | A型（序論・本論・結論型） | B型（概要・序論・本論・論議型） |
|---------|---------------------|--------------------------|
| 適した場面 | 文科系論文、従来型レポート | 理系論文、現代的レポート |
| 特徴 | 古典的・安定した構成 | 重点先行、考察セクションあり |
| 結論の位置 | 独立した「結論」節 | 本論の末尾に組み込む |
| 追加要素 | なし | 「概要（要旨）」「論議（考察）」 |

### 理系レポートの標準構成（IMRaD）

理系論文・実験レポートでは以下の構成が世界標準となっている。

| セクション | 英語名 | 内容 |
|-----------|--------|------|
| はじめに | Introduction | 研究の背景・目的・先行研究 |
| 方法 | Method | 実験・調査の方法（再現可能な程度に詳細に） |
| 結果 | Results | 得られた結果を客観的に記述 |
| 考察 | Discussion | 結果の解釈・一般化・先行研究との比較 |

加えて、**要旨（Abstract）** を論文の冒頭に置くことが一般的。要旨は「何を行って、どんな結果が得られたかの概要」を200〜400字程度で記述する。

### 日本語と英語の構成の違い

日本語の文章構成は英語と異なる傾向がある。

| 特徴 | 日本語的傾向 | 英語・学術的標準 |
|------|------------|----------------|
| 主張の提示 | 結論を最後に述べる（起承転結） | 冒頭に主張を述べる（重点先行） |
| 中心文の位置 | パラグラフ末尾に置きやすい | パラグラフ冒頭に置く |
| 文の構造 | SOV型（述語が文末） | SVO型（述語が前半） |

**学術文章では、日本語の慣習より英語・学術的標準の「重点先行」を採用することが推奨される。**

### 起承転結はレポートに不向き

起承転結は漢詩の表現形式を散文に借用したもので、文学的効果を狙う場合に適している。レポート・論文は「調査によって得た事実を積み上げ、それにもとづいて自分の考えをまとめて書くもの」であり、読者の心情に訴えることは厳に慎まなければならない。したがって、**レポートの構成として起承転結を用いることは適当でない**。
