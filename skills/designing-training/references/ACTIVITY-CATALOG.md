# アクティビティカタログ

研修で活用できるアクティビティの総合カタログ。シーン別（オープニング・クロージング・リビジット・エナジャイザー）に分類し、実践に必要な情報を一覧できるように整理。

---

## 使い方

各アクティビティは以下の形式で記述している：

| 項目 | 内容 |
|------|------|
| **目的** | このアクティビティで達成したいこと |
| **対象人数** | 推奨する参加者数の範囲 |
| **時間** | 目安所要時間 |
| **準備物** | 事前に用意が必要なもの |
| **手順** | ステップバイステップの進め方 |
| **メリット** | このアクティビティを選ぶ理由 |
| **バリエーション** | 応用・アレンジ方法 |

---

## 1. オープニングアクティビティ

### 目的と位置づけ

オープニングでは「研修内容と関連性があること」が最重要。参加者の最大の関心事を打ち破り、研修内容に意識を集中させる。人は**最初と最後を記憶しやすい**ため、オープニングには大切なメッセージを組み込む。

---

### A-1. 視点を変える

| 項目 | 詳細 |
|------|------|
| **目的** | 「ものの見方・視点を変えよう」というオープニングメッセージを体験で伝える |
| **対象テーマ** | どのテーマでも可（特に「新しい視点」を求める研修） |
| **対象人数** | 何人でも可 |
| **時間** | 3分 |
| **準備物** | 錯視図（フリップチャートに描くか印刷して貼り出す） |

**手順**
1. 「この線の長さは同じですか？違いますか？」と問いかける
2. 「違う」という回答が返ってくることを想定
3. 「同じ長さだと言えるような見方はできますか？」と重ねる
4. 高速道路の柱など日常的な例を使って解説
5. 「今日はこのように新しい視点を持って学んでいただければ」と研修につなげる

**メリット**
- 言葉だけではなく体験を通じてポイントが伝わる
- だまし絵など図を変えれば何度も使い回せる

**バリエーション**
- ミュラー・リヤー錯視、ルビンの壺など様々な錯視図に差し替え可能
- 同一の図は1回限り（使い回すと驚きがなくなる）

---

### A-2. できるだけ高くへ

| 項目 | 詳細 |
|------|------|
| **目的** | 「自分の中に固定観念がある」ことを体験させ、ストレッチ目標への意欲を高める |
| **対象テーマ** | どのテーマでも可（特に目標設定・自己成長系） |
| **対象人数** | 何人でも可（5〜6人チーム編成） |
| **時間** | 5分 |
| **準備物** | フリップチャートまたは模造紙（肩の高さに下の線がくる位置に壁へ貼る）・太いペン（1人1本） |

**手順**
1. チームから代表者1人が前に出る（全体10人以下なら全員でも可）
2. ペンを持って「できるだけ高い位置に横線を1本描いてください（ジャンプ禁止）」と指示
3. 全員が描いたら：「今度は先ほどより1センチでも高い位置に線を描いてください」
4. 2回目の線がどれだけ1回目より高かったか確認
5. 「何を工夫しましたか？」と問いかけ、気づきを引き出して研修テーマへとつなげる

**メリット**
- 動きがあり言葉だけより納得感が高い
- 全員に成功体験を積ませることができ、場が和む
- 創意工夫の余地があり、クリエイティビティが刺激される

**バリエーション**
- チームで代表者1人が行う場合、残りは「応援団」として声援を送る
- 線が引けない場合は付箋を高い位置に貼る方法に変更可
- エナジャイザーとしても活用可

**注意**
- 身体的な制限がある参加者への配慮が必要
- 「人と比べず、自分が前回より高く」という点を強調する

---

### A-3. 最高の経験・最悪の経験

| 項目 | 詳細 |
|------|------|
| **目的** | 参加者自身の経験から研修内容への共感を引き出し、主体性を高める |
| **対象テーマ** | どのテーマでも可 |
| **対象人数** | 何人でも可（5〜6人チーム） |
| **時間** | 8〜12分 |
| **準備物** | フリップチャート（「B」「W」と書いておく） |

**手順**
1. チームのリーダーを決める（早起きの人など）
2. リーダーが「B（ベスト）」か「W（ワースト）」を早い者勝ちで選択
3. Bチームは「良かった経験」、Wチームは「良くなかった経験」をメンバーで共有（5分）
4. 各チームのキーワードを全体で発表し、フリップチャートに書き留める
5. 発表されたキーワードを研修コンテンツへつなげる

**メリット**
- 自分の経験を肯定してもらえる場になる
- 「自分が口にしたこと」なので受け入れやすく、反論が出にくい
- 第三者の話にすることで、客観的かつ本音が引き出しやすい

**バリエーション**
- リーダーシップ研修：「最高/最悪のリーダー」
- 接客研修：「最高/最悪の接客体験」
- チームワーク研修：「チームワークが良かった/悪かったチーム」

**注意**
- 必ず「自分自身」ではなく「第三者（受けた接客、過去の上司など）」の話にさせる
- 最初は具体的エピソードを話してもらい、その後キーワードを引き出す（最初からキーワードを求めると表面的な回答になりがち）

---

### A-4. 意外性のある発言

| 項目 | 詳細 |
|------|------|
| **目的** | 重要なメッセージを意外性ある事実で印象づけ、興味を引きつける |
| **対象テーマ** | どのテーマでも可 |
| **対象人数** | 何人でも可 |
| **時間** | 2分程度 |
| **準備物** | 意外性のある統計データ・事例・事実 |

**手順**
1. その日の研修テーマに深く関わる、知られていない事実や統計を準備
2. クイズ形式で「○○は何パーセントだと思いますか？」と問いかける
3. 答えを発表し解説する
4. 研修のテーマ・目的へとつなげる

**メリット**
- 興味を引き、意外性があれば印象に残る
- 動きを伴わないため大人数でも活用しやすい

**バリエーション**
- クロージングや研修途中の興味喚起にも活用可

**注意**
- 数字の根拠・出典を確認され場合に備えて準備しておく

---

### A-5. 「学びたいこと」投票

| 項目 | 詳細 |
|------|------|
| **目的** | 参加者が自ら「何を学びたいか」を明確にし、主体性を高める |
| **対象テーマ** | どのテーマでも可 |
| **対象人数** | 何人でも可（20人程度まではシール直接、それ以上はデジタルツール推奨） |
| **時間** | 8〜10分 |
| **準備物** | カラーシール（1人3〜4枚）・トピック一覧を書いたフリップチャート |

**手順**
1. 研修前に「今日学ぶトピック一覧」フリップチャートを貼り出す
2. 参加者に「興味があるトピック3つにシールを貼ってください」と依頼
3. 全員貼り終えたら各トピックの得票数を確認
4. アジェンダと関連づけ「どこに時間を使うか」の説明に活用

**メリット**
- 参加者のニーズをその場で把握できる
- 「これを学びたい」と意思表示することで主体性が高まる
- 立ってシールを貼りに来る動きが脳の活性化につながる

**バリエーション**
- 色を変えて「理解度（青：知っている、赤：知らない）」の確認にも使用可
- クロージングで「最も大きな学びは何か」に活用可

---

### A-6. 「何を」「どうして」「どのように」

| 項目 | 詳細 |
|------|------|
| **目的** | 参加者が学習の目的意識（What/Why/How）を自分の言葉で整理する |
| **対象テーマ** | どのテーマでも可 |
| **対象人数** | 何人でも可（チーム単位でも可） |
| **時間** | 10分 |
| **準備物** | 付箋（1人3枚以上）・ホワイトボードまたはフリップチャート（What/Why/How記入済み）・太いペン |

**手順**
1. 研修内容紹介後、参加者に付箋を3枚配布
2. 各付箋に「What（最も学びたいこと）」「Why（なぜそれが重要か）」「How（どう学びたいか）」を記入
3. 書き終えたらホワイトボードの該当欄に貼る
4. 貼られた内容を講師が確認し、研修デザインへの反映状況を説明

**メリット**
- 「教えられる」ではなく「自分から学ぶ」姿勢が醸成される
- Howを聞くことで参加者の学習スタイルへの配慮が可能に

---

### A-7. 「約束」と「お願い」（グラウンドルール）

| 項目 | 詳細 |
|------|------|
| **目的** | 参加者自身がグラウンドルールを決めることでオーナーシップを高める |
| **対象テーマ** | どのテーマでも可 |
| **対象人数** | 10人以上推奨（5〜6人チーム） |
| **時間** | 5〜10分 |
| **準備物** | 付箋（1人10枚程度）・太いペン・フリップチャート |

**手順**
1. チームを「Iチーム（Instructor側）」と「Pチーム（Participant側）」に分ける
2. Iチーム：講師が守るべきルールを付箋に書き出す
3. Pチーム：参加者自身が守るグラウンドルールを付箋に書き出す
4. 全員が集まって10→5個に絞る
5. 研修中3時間ごとにグラウンドルールを見直す

**メリット**
- 押しつけられた感覚が低下し、研修への参画意欲が高まる
- 講師もリクエストを受け取ることができる

---

### A-8. ユニークな点と共通点

| 項目 | 詳細 |
|------|------|
| **目的** | 自己紹介を楽しい方法で行い、チームビルディングにつなげる |
| **対象テーマ** | どのテーマでも可（対人関係スキル研修に特にフィット） |
| **対象人数** | 何人でも可（5〜6人チーム） |
| **時間** | 10〜15分 |
| **準備物** | フリップチャート（全体見本用1枚＋各チーム1枚）・太いペン |

**手順**
1. フリップチャートに大きな円を描き、中に共通点スペース、外にメンバー名を書く
2. メンバー同士で質問し合いながら「共通点3つ」を円の中に記入
3. 各自の「ユニークな点2つ」を自分の名前の下に記入
4. 完成したら全体で発表・共有
5. 気づいたことを研修テーマへとつなげる

**注意**
- 「見た目でわかるもの（男性1人だけ等）」は挙げないよう事前に伝える

---

### A-9. 5つ選ぶ

| 項目 | 詳細 |
|------|------|
| **目的** | 研修テーマに関連づけた自己紹介でアイスブレイクを行い、集中力を高める |
| **対象テーマ** | どのテーマでも可 |
| **対象人数** | 何人でも可（5〜6人チーム） |
| **時間** | 5〜10分（1人1分目安） |
| **準備物** | 15のトピックが書かれた用紙または投影資料 |

**手順**
1. 15のトピック（「好きな食べ物」「今日の研修で学びたいこと」等）を提示
2. 各参加者が5つ選んでチーム内で自己紹介
3. 研修テーマに関連するトピックを2〜3つ含めておく

**バリエーション**
- リビジット：研修内容のクイズ形式に差し替えて復習ツールとして使用
- エナジャイザー：「エベレストに登ったことがある人は？」等の珍しい体験クイズに

---

### A-10. ネームテント

| 項目 | 詳細 |
|------|------|
| **目的** | 構造化された自己紹介で参加者の緊張をほぐし、研修テーマへ関連づける |
| **対象テーマ** | どのテーマでも可 |
| **対象人数** | 何人でも可（5〜6人チーム） |
| **時間** | 10分（作成2分＋自己紹介1人1分） |
| **準備物** | A4用紙（1人1枚）・太いペン |

**手順**
1. A4用紙を2回折って三角形のネームテントを作成
2. 表面に名前を書く
3. 裏面に4つの項目（一般2つ＋研修テーマ関連2つ）を記入
4. ネームテントを見せ合いながら1人1分程度で自己紹介

**メリット**
- まず「書いてから話す」ので全員が話しやすい
- 記入内容を講師が事前にコントロールでき、研修テーマへの誘導が容易

## 2. クロージングアクティビティ

### 目的と位置づけ

クロージングの目的は「学習内容の確認」「達成感・充実感の醸成」「アクションプランの策定」の3つ。**ポジティブなトーンで終わる**よう設計し、アンケートや事務連絡を最後にしない。

推奨順序：① アンケート → ② 習得確認アクティビティ → ③ アクションプラン → ④ メッセージ性のあるコンテンツ

---

### C-1. 学びのラボ（LABB）

| 項目 | 詳細 |
|------|------|
| **目的** | 学びや気づきを整理し、実践へのコミットを形成する |
| **対象テーマ** | どのテーマでも可 |
| **対象人数** | 何人でも可 |
| **時間** | 10〜15分（個人記入5分＋共有5〜10分） |
| **準備物** | LABBワークシート（1人1枚）または白紙 |

**LABB の4要素**

| 文字 | 英語 | 記入内容 |
|------|------|--------|
| **L** | Lesson | 今日の研修からの学びや気づき |
| **A** | Action | ビジネス上の成果につなげるために実践すること |
| **B** | Barrier | 実践の障害となりそうなこと |
| **B** | Benefit | 実践することで得られそうなメリット |

**手順**
1. ワークシートを配布し個人で記入（5分）
2. ペアまたはチームで共有（5〜10分）
3. 数名に全体への発表を依頼（任意）

**バリエーション**
- 希望者に全体発表してもらい、講師が学びの要点を強調する

---

### C-2. 1ヶ月後、従業員食堂で

| 項目 | 詳細 |
|------|------|
| **目的** | 研修後の実践イメージを対話を通じて具体化する |
| **対象テーマ** | どのテーマでも可 |
| **対象人数** | 何人でも可（2〜3人ペア/グループ） |
| **時間** | 5〜10分 |
| **準備物** | 特になし |

**手順**
1. 参加者全員がペアまたは3人組に分かれる
2. 「1ヶ月後、この研修に参加した仲間に食堂でばったり会った場面」を想定
3. 個人でイメージを膨らませる（2分）
4. 「あ、お久しぶりです！あの研修どうでしたか？」から始め、実践成果を報告し合う対話を行う（5分）

**メリット**
- 書くより短時間で実践イメージが形成できる
- 即興劇的な楽しさがあり、「祝い」の場になる

---

### C-3. ギャラリーウォーク

| 項目 | 詳細 |
|------|------|
| **目的** | 研修を通じて作成したフリップチャートを見て回ることで、学習内容を総復習する |
| **対象テーマ** | どのテーマでも可 |
| **対象人数** | 30人程度まで |
| **時間** | 10〜12分 |
| **準備物** | 研修中に作成したフリップチャート（壁に貼り出し）・メモ用紙・太いペン |

**手順**
1. 研修中、主要ポイントをフリップチャートにまとめ壁に順次貼り出しておく
2. クロージングで全員がメモ用紙を持って立ち上がり、部屋中のフリップチャートを見て回る
3. アクションプランに取り入れたいことをメモに書き写す
4. ペアで語り合いながら回る（「これのポイントは？」「自分はどう実践するか？」）

**メリット**
- 動き回るのでエナジャイザー効果も兼ねる
- 複数日研修では2日目のオープニングにも活用可

---

### C-4. コンサルティング封筒

| 項目 | 詳細 |
|------|------|
| **目的** | 参加者同士がアドバイスし合い、互いのネットワーク構築につなげる |
| **対象テーマ** | どのテーマでも可 |
| **対象人数** | 何人でも可（10人以上は5〜6人チーム） |
| **時間** | 10〜15分 |
| **準備物** | 封筒（1人1枚）・カード（参加者人数分×参加者数） |

**手順**
1. **オープニングまたは前半終了時**：各自が封筒の表に「名前＋この研修で学んで帰りたいこと」を記入し、壁に貼り出す
2. 休憩時間中などに他の参加者が内容を確認できるようにする
3. **クロージング**：封筒を参加者間で回し、書かれた「学びたいこと」に対するアドバイスをカードに書いて封筒に入れる
4. 全員が自分の封筒を受け取り、中のアドバイスを確認・共有

**メリット**
- 誰がどんな知識・経験を持つか可視化されネットワーク形成につながる
- アドバイスを「贈る」ことで参加者の貢献意識が高まる

---

### C-5. お祝いカード

| 項目 | 詳細 |
|------|------|
| **目的** | 学びのアクションプランを言語化し、参加者同士が称え合う |
| **対象テーマ** | どのテーマでも可 |
| **対象人数** | 何人でも可 |
| **時間** | 7〜10分 |
| **準備物** | デザインされたカード（7cm×12cm程度）1人1枚 |

**手順**
1. 各自がカードに「名前・最も重要な学び・職場での実践方法」を記入
2. チームで1人が全員のカードを回収し、シャッフルして再配布
3. 受け取ったカードの裏に「その人の研修中の貢献」を記入
4. 「私は○○さんのカードを持っています。○○さんが実践するのは〜」と読み上げ、カードを本人に手渡す

**メリット**
- カードを持ち帰れるので覚えていられる
- 他者からの感謝の言葉が実践の後押しになる

---

### C-6. 始めること・やめること・続けること

| 項目 | 詳細 |
|------|------|
| **目的** | 新しい行動を取り入れると同時に「Unlearn（アンラーン）」を促し、行動変容を具体化する |
| **対象テーマ** | どのテーマでも可 |
| **対象人数** | 何人でも可 |
| **時間** | 5〜10分 |
| **準備物** | ワークシート（3列）1人1枚 |

**手順**
1. 個人で3つの項目に分類して書き出す（各1〜3個）
   - **始めること**：新たに取り入れる行動
   - **やめること**：手放す・変えるべき行動
   - **続けること**：今後も継続する行動
2. チーム内で共有

**メリット**
- 「やめること」を書くことでアンラーンが具体的になる
- 「続けること」で現状の自己を肯定できる

---

### C-7. 理由トップ10

| 項目 | 詳細 |
|------|------|
| **目的** | アクションプランに優先順位と理由を付けることで実践への納得度を高める |
| **対象テーマ** | どのテーマでも可（特に対人関係スキル研修向き） |
| **対象人数** | 何人でも可（5〜6人チーム） |
| **時間** | 20〜30分 |
| **準備物** | 付箋（1人5枚）・フリップチャート（チームに1枚）・太いペン |

**手順**
1. 個人で「実践しようと思うことトップ5」と理由を付箋に書く（5分）
2. チームで共有し「トップ10」に集約する
3. チームのトップ10をフリップチャートにまとめ全体で発表

**メリット**
- 理由に納得すると実践意欲が持続する
- 研修での学びを自分のこととして捉えられるようになる

---

### C-8. 引用する

| 項目 | 詳細 |
|------|------|
| **目的** | 研修内容の重要な点を著名人の言葉で印象づける |
| **対象テーマ** | どのテーマでも可 |
| **対象人数** | 何人でも可 |
| **時間** | 2分程度 |
| **準備物** | 研修内容を後押しする引用文 |

**手順**

2. 専門家・著名人の言葉を引用して信頼性を高める
3. 「皆さんも今日の学びを活かして……」と行動への期待を伝えて終了

**メリット**
- 個人の主張ではなく著名人の言葉により説得力が増す
- シンプルで大人数・短時間でも使いやすい

### C-9. オープニングに戻る

**メリット**

## 3. リビジットアクティビティ

### 目的と位置づけ

エビングハウスの忘却曲線によると1回聞いた情報の90%以上が30日以内に忘れられる。**20分ごとに1〜2分のリビジットを行う**（90/20/8の法則）ことで短期記憶から長期記憶への移行を促す。

---

### R-1. カードクイズ

| 項目 | 詳細 |
|------|------|
| **目的** | 参加者が自分でクイズを作ることで3回のリビジットを実現する |
| **対象テーマ** | どのテーマでも可 |
| **対象人数** | 何人でも可 |
| **時間** | 5〜10分 |
| **準備物** | 名刺サイズのカード（1人数枚） |

**手順**
1. 各チームに「研修内容を理解していれば答えられるクイズ」を数問作成してもらう
2. 回収し重複を除いてシャッフル
3. 他チームに配りチームで解答
4. チーム間でローテーションして全員が全カードに解答

**メリット**
- 参加者は「テキストを見直す→クイズを作る→クイズに答える」の3回リビジットできる
- カードを使うことでエナジャイザー効果も兼ねる

---

### R-2. 同じ形の友だち

| 項目 | 詳細 |
|------|------|
| **目的** | 動きを伴うクイズで全員を巻き込み、楽しみながら理解を確認する |
| **対象テーマ** | どのテーマでも可 |
| **対象人数** | 20〜30人程度 |
| **時間** | 5〜10分 |
| **準備物** | 色・形が異なるキューブ（人数分以上）・スクリーン |

**手順**
1. 多様な形・色のキューブを部屋後方に配置
2. クイズをスクリーンに投影（選択肢=「赤ハート」「青い星」等のキューブの色・形で表現）
3. 各自が答えと思うキューブを取りに行く
4. 同じキューブを取った人で集まり「なぜそれが答えか」を議論
5. 正解を発表・解説

**メリット**
- 動きがあり安心して間違えられる（ゲーム的な楽しさ）
- ネットワーキングの導入にも応用可

---

### R-3. 三角レビュー

| 項目 | 詳細 |
|------|------|
| **目的** | クイズの作成→検証→解答の3段階で研修内容を複数回リビジットし、理解を確実にする |
| **対象テーマ** | どのテーマでも可 |
| **対象人数** | 20〜30人（5〜6人チーム×3チーム以上） |
| **時間** | 30分程度（作成10分・検証10分・解答10分） |
| **準備物** | フリップチャート（チームに1枚）・太いペン |

**手順**
1. 各チームで研修内容のクイズを10問作成
2. 隣のチームに移動し、クイズの質を検証して10問→8問に絞る
3. さらに隣のチームに移動し、8問に解答する
4. 元のチームに戻り解答を採点・確認

---

### R-4. フリップチャート・レビュー

| 項目 | 詳細 |
|------|------|
| **目的** | 参加者が自分たちの言葉でポイントをまとめることで記憶に定着させる |
| **対象テーマ** | どのテーマでも可 |
| **対象人数** | 20名程度まで（ペアか少人数チーム） |
| **時間** | 12〜15分 |
| **準備物** | トピック別フリップチャート（壁に貼り出し）・太いペン |

**手順**
1. 主要トピックをタイトルとして書いたフリップチャートを壁に配置
2. ペアで部屋を回りながら各フリップチャートに学びや気づきを書き加える（8分）
3. 図式化・イラスト追加の時間をとる
4. メモ持参で再度一周し、アクションプランをメモしながら確認（5分）
5. 「〇」印のついた疑問箇所を確認・解説

---

### R-5. 運命のサイコロ

| 項目 | 詳細 |
|------|------|
| **目的** | サイコロで出たトピックについて語ることで楽しみながら復習する |
| **対象テーマ** | どのテーマでも可 |
| **対象人数** | 何人でも可（5〜6人チーム） |
| **時間** | 10分程度 |
| **準備物** | サイコロ（チームに1つ）・6トピックのリスト（フリップチャートまたは投影） |

**手順**
1. 6つのリビジットトピックに番号をつけてフリップチャートに書く
2. チームで1人ずつサイコロを振り、出た目の番号のトピックについて語る
3. タイマーをセットして時間内に順番に繰り返す

**バリエーション**
- 複数日研修では「そのトピックを実践した結果」を報告し合う場面にも活用

---

### R-6. パワー・フレーズ

| 項目 | 詳細 |
|------|------|
| **目的** | 「良いと思った言葉」を随時記録して長期記憶への移行を助ける |
| **対象テーマ** | どのテーマでも可（コミュニケーション・営業・接客で特に有効） |
| **対象人数** | 何人でも可 |
| **時間** | 研修全体を通じて運用（まとめて確認は10分程度） |
| **準備物** | 「パワー・フレーズ」掲示ポスター・付箋 |

**手順**
1. 研修序盤に「良いな、覚えておきたいと思った言葉を付箋に書いてポスターに貼ってください」と依頼
2. 研修を通じて参加者が随時付箋を貼り足す
3. 研修後半またはクロージング時に全員でポスターを振り返る

**メリット**
- 他の参加者の言葉が自分の記憶にも残る
- 賞賛し合う場になりポジティブな雰囲気が生まれる

2. タイマーをセットしてチームで取り組む
3. 全チームが書いた文章を共有・比較

**注意**

## 4. エナジャイザー

### 目的と位置づけ

集中力の低下・食後の眠気・疲れが見えた時に実施する短い活動。**研修内容と無関係でも構わない**。身体を動かすフィジカル系とゲーム・クイズを使うメンタル系の両方がある。

**実施の判断基準**

| 状況 | 判断 |
|------|------|
| 午後の研修で集中力が明らかに低下している | 実施する |
| コンプライアンス等の深刻テーマ | 慎重に判断 |
| 半日以下の研修 | 通常は不要 |

---

### E-1. そこでストップ！

| 項目 | 詳細 |
|------|------|
| **目的** | 体を動かしながら参加者同士の交流を促す |
| **対象テーマ** | どのテーマでも可 |
| **対象人数** | 何人でも可 |
| **時間** | 5分程度 |
| **準備物** | 特になし（動き回れるスペースが必要） |

**手順**
1. 参加者に「1番」「2番」を交互に割り当て2チームに分ける
2. 2列になって向かい合う
3. 全員で「あ、い、う、え、お……」と大きな声で50音を順番に言う
4. 適当なタイミングで講師が「ストップ！」と叫ぶ
5. 1番の人が2番チームの誰か1人とペアになり、指定されたトピックについて短時間話す

---

### E-2. トイレットペーパー

| 項目 | 詳細 |
|------|------|
| **目的** | 意外なアイテムで参加者の笑いと集中力を回復する |
| **対象テーマ** | どのテーマでも可 |
| **対象人数** | 何人でも可 |
| **時間** | 3〜5分 |
| **準備物** | トイレットペーパー（1ロール） |

**手順**
1. トイレットペーパーをチームに回し「必要な分だけ取ってください」と指示（用途は言わない）
2. 全員が取り終えたら「取った枚数だけ自己紹介や最近のトピックについて話してください」と伝える
3. 多く取った人ほど多く話すことになる

---

### E-3. つまようじクイズ

| 項目 | 詳細 |
|------|------|
| **目的** | 集中力を刺激する短いパズルで脳を活性化する |
| **対象テーマ** | どのテーマでも可 |
| **対象人数** | 何人でも可 |
| **時間** | 3〜5分 |
| **準備物** | つまようじ（1人5〜10本） |

**手順**
1. つまようじで図形を作って見せる（例：5本で正方形2つ）
2. 「つまようじを2本動かして正方形を1つにしてください」等のパズルを出す
3. チームで相談して答えを見つける

---

### E-4. 山手線ゲーム

| 項目 | 詳細 |
|------|------|
| **目的** | テーマに関連した言葉を素早く連想することで脳と会話を活性化する |
| **対象テーマ** | どのテーマでも可（研修テーマに関連したお題で実施可） |
| **対象人数** | 何人でも可（チーム対抗でも可） |

2. テンポよく1人ずつ言っていく（拍手でリズムを取る）
3. 詰まったら脱落またはチーム交代

---

### E-5. 整列！

|------|------|

3. 全員が無言で身振り手振りだけで情報交換して並ぶ
