# ファシリテーション基礎・原理

研修ファシリテーションの理論的基盤。ファシリテーションが必要な理由、集合研修の落とし穴、講師の役割定義、時間配分法則（90/20/8）、EATフレームワークを網羅する。

---

## 1. なぜ研修にファシリテーションが必要か

### 1-1. 研修の目的は行動変容にある

研修の目的は**ビジネス上の成果を出すこと**であり、知識を「伝える」ことは目的ではない。参加者が職場に戻って学んだことを実践し、成果を出すことが最終ゴールである。

実践につながる3つの条件：
1. 学んだ内容を**理解・習得**できていること
2. それを**記憶**していること
3. **実践しようという感情**を伴っていること

講義だけでは、行動変容はほぼ起きない。大量の情報を受動的に聞くだけでは、理解・記憶・感情の変化をすべて達成するのは難しいからである。

### 1-2. 「気づき」だけでは不十分

よく聞くような参加者のコメントに注意が必要：
- 「大変ためになりました」
- 「新たな気づきが多く有意義でした」
- 「○○の大切さに気づきました」

これらは**感想レベルに留まっている**可能性が高い。「気づく」段階はまだ抽象的であり、「具体的な行動に落とし込む」まで到達していない。

研修後に実践につながるためには、「○○が大切だとわかった」ではなく、「何をどう変えるか」の具体的な行動を定義した状態で終了する必要がある。

### 1-3. 集合研修だからこそできること

参加者が物理的に集合して行う価値がある研修とは何かを問い直す。

**eラーニングや動画で代替できること**：
- 知識インプット（一方向の情報提供）
- 定形スキルの確認

**集合研修でしかできないこと**：
- 参加者同士の対話・学び合い
- 疑似体験を通じた感情移入
- リアルタイムのフィードバックと練習
- 場の雰囲気によるモチベーション喚起

集合研修は「人がわざわざ集まるからこそ得られる成果」を生み出すことに特化すべきである。

## 2. 集合研修の典型的な落とし穴

### 落とし穴1：突然の指名で発言を求める

**問題点：**
- 突然の指名は**過度な緊張**を引き起こす
- 過度なストレス状態では脳の学習能力が低下する
- ネガティブな感情と結びついた情報は長期記憶に移行しにくい
- 「講師 対 参加者1名」という1対1の対話になり、他の参加者が蚊帳の外になる

**目指すべき形：**
講師と参加者のやり取りに加え、**参加者同士が話す**対話の形。正解を学ぶだけの場ではなく、学び合い・教え合い・ピアサポートが行われる場。

### 落とし穴2：「自由に」リフレクションさせる

「自由に感想を述べてください」「自由に話し合ってください」というインストラクションが引き起こすリスク：
- 適切な問いかけがないため、話が深まらない
- 本来の趣旨と異なる内容（デメリット、例外）に時間を費やす
- 「役に立たない」「現場は違う」などの否定的な発言を誘発する

**解決策：** 「何を目的に」「何について」「どう考えてほしいのか」を明確に示し、ゴールに向かって導くデザインにする。

### 落とし穴3：有効でない問いかけ

オープンすぎる問いかけの問題例：
- 「管理職として、これまでと最も違うことは何ですか？」（研修前に答えにくい）
- 「良いリーダーとは？」（範囲が広すぎ、講師の意図とズレた回答が出やすい）

参加者は問われたことに答えようとする。つまり「感想を述べてください」と言えば感想を、「具体的なアクションを3つ挙げてください」と言えば3つのアクションを挙げようとする。

問いかけの質と具体性を精査することが、ファシリテーションの核心となる。

## 3. ファシリテーターとしての講師の役割

### 3-1. 講師の3つの役割

1. **情報を提供する（Content Expert）**：知識・理論・スキルを伝える
2. **場を設計する（Designer）**：参加者の学びが起きるよう研修をデザインする
3. **場を進行する（Facilitator）**：参加者の主体的な学びを引き出し、導く

多くの講師が1に特化している。しかし研修の目的（行動変容→成果）を達成するためには、3が欠かせない。

### 3-2. 講師が行う3つの仕事

1. **研修のデザイン**：何をどう学んでもらうかを事前に設計する
2. **コンテンツのデリバリー**：情報を効果的に伝える
3. **ファシリテーション**：参加者の理解・気づき・実践意欲を引き出す

「ステージ上の賢者」から「ステージ脇で導く役割」への転換が求められる。

### 3-3. ファシリテーターとしての行動原則

| 行動 | 概要 |
|------|------|
| 安心できる場をつくる | 心理的安全性を確保し、学習への集中を促す |
| 問いかけを設計する | 目的に沿った問いで参加者の思考を導く |
| 参加者同士の対話を促す | 講師対参加者の1対1から、参加者同士の学び合いへ |
| 経験を引き出す | 参加者の過去の経験・知識を活用して学びを深める |
| 行動に落とし込む | 「気づき」を「具体的なアクション」に変換する |
| 準備を徹底する | 問いかけ・インストラクション・タイミングを事前に設計する |

## 4. 参加者の主体性を引き出す基本原理

### 4-1. 準備の重要性

ファシリテーションが難しいと感じられる主な理由は「出たとこ勝負」になりがちだから。しかし、基盤となる原理原則を知ったうえで事前設計をすることで、安定したパフォーマンスを発揮できる。

**事前に設計すべきもの：**
- 問いかけの内容と具体的な文言
- アクティビティの目的と進め方のインストラクション
- 参加者から想定される反応とその対応
- 各アクティビティの所要時間とバッファ

**原理原則に則ったファシリテーションは再現性がある。** 経験やスキルへの依存度を下げ、誰でもある程度の品質を発揮できるようになる。

### 4-2. 準備チェックリスト

研修前に確認すべき項目：

**コンテンツ（WHAT）の設計：**
- [ ] 研修の最終ゴール（行動変容・成果）を明確にした
- [ ] 各トピックの到達目標を設定した
- [ ] 必要な知識・スキルのリストを整理した

**ファシリテーション（HOW）の設計：**
- [ ] 各トピックのオープニングアクティビティを設計した
- [ ] 8分に1回の参画（問いかけ）を配置した
- [ ] 20分ごとのリビジット（振り返り）を設計した
- [ ] クロージングのアクションプラン記入時間を確保した
- [ ] 参加者から想定される反応とその対応をシミュレーションした
- [ ] 問いかけの文言を具体的に書き出した

**環境・場の設計：**
- [ ] 会場レイアウトを決定した（推奨：円卓型or島型）
- [ ] グラウンドルールを準備した
- [ ] 時間管理ツール（タイマー表示等）を準備した

## 5. 90/20/8の時間配分法則

### 5-1. 法則の概要

学習効果に基づいた研修の時間配分原則：

| 単位 | 内容 | 根拠 |
|------|------|------|
| **90分** | 1ブロックの最大時間 | 大人が理解力を保って集中できる上限 |
| **20分** | コンテンツの最小単位 | 短期記憶に保持しながら聞ける時間 |
| **8分** | 能動的参加の間隔 | 受け身状態が8分を超えると退屈・意識散漫が始まる |

### 5-2. 20分ブロックの構成要素

1ブロック（20分）は以下の3要素で構成する：

| 要素 | 役割 | 例 |
|------|------|-----|
| **コンテンツ** | 知識・理論の提供 | 講義・説明・デモ |
| **参画** | 能動的な関与 | 問いかけ・ペア対話・個人ワーク |
| **リビジット** | 振り返り・記憶定着 | まとめ・クイズ・整理 |

### 5-3. 90分ブロックの組み立て例

```
[0:00-0:03]  トピックのオープニング（問いかけ/アクティビティ）
[0:03-0:08]  コンテンツA（講義・説明）
[0:08-0:10]  参画①（問いかけ→個人考察）
[0:10-0:18]  コンテンツB（続き）
[0:18-0:20]  参画②（ペア対話）
[0:20-0:22]  リビジット①（整理・振り返り）
... （以降繰り返し）
[1:27-1:30]  クロージング（アクションプラン）
[1:30]       休憩
```

### 5-4. 90/20/8のメリット

- **シームレスな研修**：講義とディスカッションを分断しない自然な流れ
- **受け身状態の防止**：8分ごとに能動的参加を促すため集中が持続する
- **記憶定着**：20分ごとのリビジットで短期記憶を長期記憶に変換
- **ファシリテーターとしての自然な役割発揮**：コンテンツ中も問いかけを通じてファシリテーターの役割を担う

## 6. EAT構成（Experience-Awareness-Theory）

### 6-1. EATとは

研修の構成順序に関するフレームワーク。従来の「理論→気づき→経験」（TAE）の順ではなく、「**経験→気づき→理論**」の順に構成する。

| フェーズ | 内容 | 講師の主な役割 |
|---------|------|--------------|
| **E（Experience）** | 体験・疑似体験・過去の経験引き出し | ファシリテーター |
| **A（Awareness）** | 気づきを引き出し、まとめる | ファシリテーター |
| **T（Theory）** | 理論・フレームワーク・補足情報の提供 | コンテンツ提供者 |

### 6-2. EAT構成の具体例（接客スキル研修の場合）

1. **E**：過去にお客の立場で良い/悪い接客を受けた経験を思い出してもらう
2. **A**：チームで経験をシェアし、「良い接客」「良くない接客」の要素を挙げる
3. **T**：各チームに発表してもらい、講師が補足・解説する

### 6-3. EATを選択すべき場面

```
AskUserQuestion:
- 参加者の知識レベルが均一でない場合 → EAT（Eで底上げ可能）
- 講師の「べき論」への抵抗が予測される場合 → EAT（自分たちの経験から積み上げ）
- 参加者の主体性を最大化したい場合 → EAT
- 知識習得を効率的に行いたい場合 → TAE（従来型）も有効
```

### 6-4. EATの主なメリット

- 参加者の知識レベルがバラバラな際、体験を通じて底上げできる
- 自身の体験をもとに説明を聞くため、納得度が高まる
- 過去の体験やもっている知識と関連づけて新しいことを学べるため整理しやすい
- 最初に体験があるため、興味・主体性が高まる
- 「既に知っていることを説明される」というストレスがなくなる
- 的を絞った講義ができるため、研修全体の時間短縮につながる
- 押しつけられる感覚が減り、講師の話が受け入れやすくなる

## 7. 参加者主体の研修手法の概要

### 7-1. 基本コンセプト

参加者が研修中、**受け身の状態に終始しない**ようにする。自ら考え、理解を深め、活用を検討し、スキルを練習し、職場実践の計画をするという主体的な学びを実現する。

この手法では講師は「知識伝達者」ではなく、「場を通じてビジネス成果に導く存在」として機能する。

### 7-2. 参加者主体の研修が実現する状態

| 従来型（講義中心） | 参加者主体型 |
|------------------|------------|
| 参加者は受け身で聞く | 参加者が考え・話し・書く |
| 記憶定着が困難 | 繰り返しのリビジットで定着 |
| 行動変容が起きにくい | 具体的アクションプランを立案 |
| 講師 ↔ 参加者の1対1 | 参加者同士の学び合い |
| 講師のスキル依存 | 原理原則に基づく再現性 |

### 7-3. 成人学習との接続

成人は子どもと異なり、豊富な経験と知識をすでに持っている。効果的な研修はその**既存の経験・知識を最大限に活用**して新たな学びを積み上げる設計にすべきである。

また、成人は「なぜ学ぶのか」という意義を自ら納得していなければ動機が湧かない。オープニングでの「研修の意義の自分ごと化」が重要である。
