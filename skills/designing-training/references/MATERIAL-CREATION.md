# 研修資料作成ガイド

研修で使用する各種資料（スライド・ワークシート・配付資料・ファシリテーターガイド）の
設計・作成方法と、オンライン研修における注意点を提供するリファレンス。

---

## 1. 研修資料の種類と役割

研修資料は「誰が・いつ・何のために使うか」によって明確に役割が異なる。

| 資料の種類 | 使用者 | タイミング | 目的 |
|-----------|-------|----------|------|
| **スライド（講義用）** | 講師（参加者も閲覧） | 研修中 | コンテンツの視覚的提示 |
| **ワークシート（演習用）** | 参加者 | 研修中のアクティビティ | 考える・書く・実践する |
| **配付資料（参照用）** | 参加者 | 研修後 | 研修内容の復習・参照 |
| **ファシリテーターガイド（講師用）** | 講師 | 研修中 | 進行・タイミング・話し方の指示 |

> **AskUserQuestion（実装時の判断）：** どの資料が必要かは研修の形式・目的によって異なる。以下の判断テーブルを参照。

### 資料の必要性 判断テーブル

| 研修の特徴 | 推奨資料 |
|-----------|---------|
| 知識インプット中心 | スライド + 配付資料 |
| スキル演習・ワーク中心 | スライド + ワークシート |
| 対話・ディスカッション中心 | スライド（最小限）+ ワークシート |
| オンライン研修 | デジタルワークシート（印刷不要） |
| 複数回にわたる研修 | ワークブック（全回分を1冊に統合） |
| 講師が複数いる研修 | ファシリテーターガイド（必須） |

## 2. スライド（講義用）の作成ガイド

### 2.1 スライドの役割を正しく理解する

スライドは講師の「台本」ではなく、参加者の理解を助けるための「視覚補助ツール」。

**スライドに書くべきこと：**
- キーワード・短い文章（箇条書き）
- 図・グラフ・表（視覚的に説明できるもの）
- ポイントを強調する言葉
- アクティビティの指示文

**スライドに書かないこと：**
- 講師が話す内容をすべて文章で書く（読み上げる必要がない）
- 長い文章（参加者が読み切れない）
- 1枚に多くの情報を詰め込む

### 2.2 情報量の適正化

**1スライドの情報量の目安：**

| 要素 | 推奨 |
|------|------|
| 1スライドのメッセージ | 1つに絞る |
| 箇条書きの数 | 3〜5点まで |
| 文字量 | 参加者が10秒以内に読める量 |
| フォントサイズ | 最小24pt以上（会場後方・小画面から見えること） |

**スライドの情報量が多すぎるサイン：**
- 参加者がスライドを読んでいて講師の話を聞いていない
- スライドの切り替えを急いでしまう
- 「ここに書いてある通りです」と言わなければならない

### 2.3 ビジュアルの活用

**効果的なビジュアル活用：**

| ビジュアル | 使用場面 |
|-----------|---------|
| 図・フロー図 | 手順・プロセスを示す |
| 表・マトリクス | 比較・分類を示す |
| グラフ | 数値の変化・比較を示す |
| 写真・イラスト | 感情・雰囲気・具体的なイメージを伝える |
| アイコン | 箇条書きをわかりやすくする |

**ビジュアルを使う際の注意点：**
- 飾りのためのビジュアルは使わない（目的のないビジュアルは逆効果）
- カラーは3〜4色以内に絞る（メインカラー・アクセントカラー・背景）
- 白黒印刷でも読めるコントラストを確保する

### 2.4 参加者主体に合わせたスライド構成

参加者主体の研修では、スライドを「講師が説明するための資料」から「参加者が考えるきっかけとなるもの」にシフトする。

**講義型スライドの例：**
```
❌ 「コミュニケーションの3つの要素：
    1. 言語（7%）
    2. 声のトーン（38%）
    3. 表情・ジェスチャー（55%）」
```

**参加者主体型スライドの例：**
```
✅ 「相手に与える印象の7%・38%・55%は何だと思いますか？」
   → 参加者にチャットや投票で予測させてから答えを出す
```

### 2.5 EAT構成とスライドの対応

| EATの段階 | スライドの内容 |
|---------|--------------|
| E（経験を引き出す） | 「〇〇の経験を思い出してください」「A・B・Cのどれだと思いますか？」 |
| A（気づきを深める） | グループワークの指示、問いかけの文章 |
| T（理論を提示する） | フレームワーク・概念図・まとめ |

## 3. ワークシート（演習用）の作成ガイド

### 3.1 ワークシートの目的

- 参加者が「考える・書く・整理する」ための記録媒体
- 研修後の参照・実践の記録としても使える
- 講師が一人ひとりと対話できないオンライン研修では特に重要

### 3.2 個人ワーク用ワークシートの設計

**設計のポイント：**
- 何を記入するかを明確にする（曖昧な問いは参加者が困惑する）
- 記入スペースは十分に確保する
- 時間内で書ける量に収める

**記入スペースの目安：**

| アクティビティの時間 | 書き込む量の目安 |
|------------------|---------------|
| 1分 | 1〜2文（30〜50字） |
| 3分 | 3〜5点の箇条書き |
| 5分 | 段落形式で1〜2段落 |

**効果的なワークシートの問いかけ例：**

```
個人振り返りシートの例：
─────────────────────────────
1. 今日の研修で最も印象に残ったことは何ですか？
   （1〜2つ、具体的に）
   →

2. 職場に戻ってすぐに実践したいことは何ですか？
   （具体的な行動として）
   →

3. 実践する際に予想される障壁は？ どう乗り越えますか？
   →
─────────────────────────────
```

### 3.3 グループワーク用ワークシートの設計

**グループワークシートに必須の要素：**
- グループ名・参加者名を記入する欄
- 課題・問いかけの文章（インストラクション）
- 各自が記入するスペース（個人→全体の流れが多い）
- グループとしてのアウトプットをまとめるスペース
- 発表準備のための記入欄（発表する場合）

**グループワークシートの例：**
```
グループ：___________
メンバー：___________

【課題】 このケースにおいて、あなたならどう対応しますか？

個人で考える（2分）→ メンバーと共有（3分）→ グループの意見をまとめる（2分）

■ 個人の意見（各自記入）
→

■ グループとしての意見（まとめる）
→

■ 発表用ポイント（3点以内）
1.
2.
3.
```

### 3.4 振り返りシートの設計

研修終了時・研修後のフォローアップ用の振り返りシート。

**振り返りシートに含める要素：**

| 要素 | 内容例 |
|------|-------|
| 学んだこと | 「今日の研修で新しく学んだこと・再確認したことは？」 |
| 気づき | 「これまでの自分の行動を振り返って気づいたことは？」 |
| アクションプラン | 「研修後、具体的に行動することは？（期限を含めて）」 |
| 実践の宣言 | 「誰に・何を・いつまでに実践するか」 |
| 振り返り確認 | 「〇日後に実践状況を確認する」 |

## 4. 配付資料（参照用）の作成ガイド

### 4.1 配付資料の役割

研修で学んだことを、後から参照・活用するための資料。スライドとは別に設計することが多い。

**配付資料の種類：**

| 種類 | 内容 | 使用タイミング |
|------|------|--------------|
| 要約資料 | 研修のキーポイントをコンパクトにまとめたもの | 研修後の復習 |
| チェックリスト | 行動変容のための確認リスト | 職場での実践時 |
| テンプレート | 習得したスキルを使うための型 | 職場での実践時 |
| 参考資料リスト | さらに深く学ぶための書籍・URL等 | 自主学習時 |

### 4.2 配付資料の設計原則

- **スライドをそのまま配付しない：** スライドは視覚補助ツールなので、文字が少なく、配付資料として使えない場合が多い
- **自立した資料にする：** スライドを見なくても内容が理解できる資料にする
- **実践に使える形式にする：** チェックリスト・テンプレートなど、職場で使える形式にする

### 4.3 オンライン研修での配付方法

| 方法 | 特徴 |
|------|------|
| 事前メールで送付 | 研修当日に手元にある。印刷して参加してもらえる |
| チャットでリンク共有 | 研修中に配布。URLを共有して即時アクセス |
| クラウドストレージ共有 | Googleドライブ等で共有。ファイル管理が容易 |
| デジタルワークシート | PDF/スプレッドシートなど。デジタルで記入可能 |

## 5. ファシリテーターガイドの作成方法

### 5.1 ファシリテーターガイドとは

講師（ファシリテーター）が研修を進行するための詳細な手引き書。複数の講師が同じ研修を行う場合や、講師が自分以外の人が作成した研修を担当する場合に特に重要。

### 5.2 ファシリテーターガイドに含める要素

**必須セクション：**

```
1. 研修の概要
   - 研修目的・目標
   - 対象者・前提条件
   - 所要時間
   - 必要な準備物

2. タイムテーブル
   - 時刻（開始からの経過時間）
   - 各セクションの内容
   - 使用するスライド番号
   - ツール・機能

3. 各セクションの詳細指示
   - 話し方メモ（キーワード・言い回し）
   - アクティビティの指示文（完全版）
   - 参加者の反応例と対応方法
   - 注意事項・トラブルシューティング

4. アクティビティのインストラクション
   - 8ステップの手順
   - グループ分けの方法
   - 時間設定

5. よくある質問と回答例
```

### 5.3 タイムテーブルの書き方

| 時間（経過） | 時刻 | セクション | 内容 | スライド | ツール | 備考 |
|------------|------|----------|------|---------|-------|------|
| 0:00 | 13:00 | ソフトオープニング | クイズをスライドに表示 | #1 | チャット | 3分前から開始 |
| 0:03 | 13:03 | オープニング | クイズの答えを発表 | #2-3 | 投票 | 全員が参加できるよう確認 |
| 0:08 | 13:08 | セクション1（C） | 〇〇の概念説明 | #4-7 | - | 4分で説明 |
| 0:12 | 13:12 | セクション1（S） | グループディスカッション | #8 | ブレイクアウト | 3-4人 / 3分 |
| 0:15 | 13:15 | セクション1（R） | リビジット | #9 | チャット | 1分 |

### 5.4 話し方メモの書き方

研修で話す内容を完全な台本として書くのではなく、キーワードと流れを書き留める。

**話し方メモの例：**
```
【スライド #4：〇〇の3つのポイント】
・「では、〇〇について3つのポイントをご紹介します」
・ポイント1を示す → 「まず1つ目は〇〇です。これは...」
・事例を1つ紹介（△△の事例）
・参加者への問いかけ：「みなさんの職場ではいかがでしょうか？チャットに入力してください」
・回答を2〜3つ紹介してから次へ
```

## 7. オンライン研修資料の特有の注意点

### 7.1 スライドのオンライン最適化

オンライン研修では「画面共有」でスライドが表示される。以下の点に注意。

| 課題 | 対策 |
|------|------|
| 文字が小さくて見えない | フォントサイズを最小28pt以上にする |
| 画面の一部しか見えない | 重要な要素を中央寄りに配置する |
| コントラストが低い | 背景と文字のコントラストを高める（白背景に黒文字が最も安全） |
| 複数の画面（スライド + カメラ映像）が混在 | 情報を整理し、シンプルにする |

### 7.2 デジタルワークシートの設計

オンライン研修では、印刷したワークシートを配布できない。以下の代替手段を設計する。

| 手段 | 特徴 | 使用場面 |
|------|------|---------|
| チャットへの記入 | 手軽。全員が同時に入力できる | 短い回答・1〜2文 |
| オンラインホワイトボード（Miro等） | 付箋・図を使った自由な表現 | ブレインストーミング・整理 |
| スプレッドシート（Google Sheets等） | 共同編集。構造化された記入が可能 | グループワーク・まとめ |
| PDFワークシート | 事前送付して記入してもらう | 個人ワーク・振り返り |
| 投票・アンケート機能 | 選択形式。集計が自動 | 理解度確認・意見収集 |

### 7.3 チャットを活用した「書く」活動

チャットは最もシンプルで参加ハードルが低いツール。効果的に活用する方法：

```
● アクティビティ中は「送信ボタンを押すのを待つ」指示を出す
  → 全員が同時にチャットに入力して一斉送信することで、一人の回答に引きずられない

● チャットに書いた回答を画面に映しながら紹介する
  → 多くの回答を可視化して「自分の意見が伝わった」実感を与える

● 「感想はチャットに、質問はチャットの頭に【質問】とつけて」などのルールを決める
  → 内容の分類をしやすくする
```

## 8. 資料作成の品質チェックリスト

### スライドチェックリスト
```
□ 1スライド1メッセージになっているか
□ フォントサイズは28pt以上か（オンライン・見やすさ）
□ 1スライドの情報量が多すぎないか（5点以内の箇条書き等）
□ 不要なビジュアル・装飾がないか
□ 参加者が「考える・反応する」きっかけになるスライドが含まれているか
□ スライドの番号が振られているか（「スライド5を見てください」と言えるか）
```

### ワークシートチェックリスト
```
□ 何を記入するかが明確か
□ 記入スペースは十分か
□ 時間内に記入できる量か
□ 研修後に「見返したくなる」内容か
□ アクションプランに結びつく問いかけがあるか
□ デジタル対応（オンライン研修の場合）ができているか
```

### ファシリテーターガイドチェックリスト
```
□ 研修の全体の流れが一目でわかるか
□ 各アクティビティのインストラクションが完全に記載されているか
□ タイムテーブルが詳細に記載されているか
□ よくある質問と回答が含まれているか
□ オンライン研修の場合、使用ツール・操作手順が記載されているか
```
