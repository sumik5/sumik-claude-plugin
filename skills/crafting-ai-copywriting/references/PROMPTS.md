# 15のプロンプトテクニック詳細

このドキュメントでは、各テクニックのプロンプトテンプレート、心理的背景、出力例の比較、効果的な使用場面、注意点を記載します。

## テクニック①: 禁止命令（カリギュラ効果）

### 心理的背景

**カリギュラ効果**: 人は禁止されると逆にやりたくなる心理。ローマ皇帝カリギュラの逸話に由来し、「ダメと言われるほど好奇心が刺激される」現象。

マーケティングでは「限定」「閲覧注意」「〇〇な人は見ないでください」という形で頻繁に利用される。

### プロンプトテンプレート

```
「〇〇」についてのキャッチコピーを作ってください。
ただし、「〇〇するな」「〇〇しないでください」「〇〇禁止」のような禁止命令の形にしてください。
人は禁止されると逆にやりたくなる心理があるので、それを利用したコピーにしてください。
10個お願いします。
```

### 出力例の比較

| 通常の出力 | テクニック適用後 |
|-----------|----------------|
| ブログアクセスを劇的に増やす5つの秘訣 | このアクセスアップ術、ライバルには絶対教えないで |
| 読まれるタイトルの作り方を完全解説 | そのタイトル、今すぐ使うのやめて |
| 初心者向けブログ運営ガイド | 初心者は読まないでください（本気の人だけどうぞ） |

### 効果的な使用場面

- 情報商材、ノウハウ系コンテンツ
- ターゲットを絞り込みたいとき（「本気の人だけ」等）
- SNSでのバズ狙い（好奇心を強制的に刺激）

### 注意点

- やりすぎると「釣りタイトル」と判断される
- 中身が伴わないと離脱率が上がる
- 企業の公式コンテンツには不向きな場合がある（カジュアルすぎる印象）

## テクニック②: 数字で具体性

### 心理的背景

**具体性の原理**: 人間の脳は曖昧な情報より具体的な数字を信頼する。「たくさん」「すぐに」より「97%」「3日で」の方が記憶に残り、行動を促す。

また、奇数（3、7等）や端数（97%、1,834円）は「正確に測定した」印象を与え、信頼性が増す。

### プロンプトテンプレート

```
「〇〇」についてのキャッチコピーを10個作ってください。
必ず具体的な数字を入れてください。
例：「3日で」「97%が知らない」「1万円節約」など。
数字はできるだけ意外性のあるものにしてください。
```

### 出力例の比較

| 通常の出力 | テクニック適用後 |
|-----------|----------------|
| 短時間でも稼げる副業術 | 1日30分でも稼げる副業術 |
| 多くの人が成功している方法 | 10人中8人が成功している方法 |
| すぐに結果が出る副業 | 3日で結果が出る副業の始め方 |

### 効果的な使用場面

- Before/Afterを示したいとき（ダイエット、収益改善等）
- 信頼性を高めたいとき（統計的根拠を示唆）
- 時間・金額のハードルを明示したいとき

### 注意点

- 根拠のない数字は使用禁止（誇大広告・景品表示法違反のリスク）
- 大きすぎる数字は逆に信頼を失う（「100%成功」等）
- 競合調査で使われている数字と重複すると埋もれる

## テクニック③: 問いかけ（自分事化）

### 心理的背景

**自分事化**: 質問形式にすると、読み手は無意識に「自分に当てはまるか?」を考え始める。受動的な情報受信から能動的な思考へ切り替わり、コピーへの関与度が上がる。

特に「ドキッとする質問」は、読み手の悩みや欲求を言語化し、「これは自分のための情報だ」と認識させる。

### プロンプトテンプレート

```
「〇〇」についてのキャッチコピーを10個作ってください。
すべて「疑問形」にしてください。
読み手が「ドキッ」とするような、自分のことを言われているような問いかけにしてください。
```

### 出力例の比較

| 通常の出力 | テクニック適用後 |
|-----------|----------------|
| 無理せず貯金を増やす方法 | 『今月こそ貯める』って、毎月言ってませんか? |
| お金が貯まる家計簿のつけ方 | その家計簿、つけるだけで満足してませんか? |
| 貯金を継続するコツ | なぜあなたの貯金は3日で挫折するのか? |

### 効果的な使用場面

- 悩み解決系コンテンツ（ダイエット、転職、お金の悩み等）
- ターゲットの「あるある」を言語化したいとき
- ランディングページのヘッドライン

### 注意点

- 質問が曖昧だと効果が薄い（「貯金したいですか?」→ 当たり前すぎる）
- ネガティブすぎる質問は読み手を不快にする可能性
- 解決策を提示しない質問だけのコピーは不完全

## テクニック④: 意外性（常識の逆）

### 心理的背景

**認知的不協和**: 自分の信念と矛盾する情報に触れると、人は「どういうこと?」と強い関心を持つ。この心理的緊張を解消したい欲求がクリックや読了を促す。

「常識を疑え」型のコピーは飽和した市場でも目立ちやすい。

### プロンプトテンプレート

```
「〇〇」についてのキャッチコピーを10個作ってください。
一般的な常識とは「逆」のことを主張するコピーにしてください。
「普通はAだと思われているけど、実はBだ」という構造にしてください。
```

### 出力例の比較

| 通常の出力 | テクニック適用後 |
|-----------|----------------|
| 我慢せずに貯金を増やす方法 | 『節約すれば貯まる』が、実は一番貯まらない理由 |
| 無駄遣いを減らす家計術 | 無駄遣いを減らすほど、お金が逃げる法則 |
| 効率的な貯金テクニック | 貯金しようとするな。お金は勝手に貯まる |

### 効果的な使用場面

- 競合が多いジャンル（差別化したいとき）
- 固定観念が強いテーマ（ダイエット、勉強法等）
- 「新しい視点」を提供したいとき

### 注意点

- 常識の逆を主張する以上、本文で論理的な根拠が必須
- 単なる逆張りは反感を買う（「努力は無意味」等）
- 専門性のない分野で使うと信頼を失う

## テクニック⑤: 感情ワード

### 心理的背景

**感情的共鳴**: 人間の意思決定は「理屈3割、感情7割」と言われる。感情を刺激するワード（衝撃、崩壊、禁断等）は記憶に残りやすく、行動を促す。

ただし、過度な使用は「煽り記事」と判断され、信頼を損なうリスクがある。

### プロンプトテンプレート

```
「〇〇」についてのキャッチコピーを10個作ってください。
以下の感情ワードのいずれかを必ず含めてください：
「衝撃」「激変」「禁断」「本音」「暴露」「崩壊」「覚醒」「究極」「絶望」「希望」。
感情が動くような、インパクトのあるコピーにしてください。
```

### 出力例の比較

| 通常の出力 | テクニック適用後 |
|-----------|----------------|
| 英語上達3つのポイント | 英語上達3つのポイントが、衝撃的すぎた |
| 失敗しない英語学習法 | これまでの常識が完全崩壊、失敗しない英語上達法 |
| おすすめ英語教材 | 禁断の英語教材。これ使ったら他に戻れない |

### 効果的な使用場面

- SNSでのバズ狙い（感情的反応を誘発）
- 情報商材、自己啓発系
- ニッチな専門情報（「業界の本音」等）

### 注意点

- **最も煽りっぽくなりやすいテクニック**
- 複数の感情ワードを詰め込むと逆効果（「衝撃!禁断の究極暴露!」→ 信頼ゼロ）
- 中身が伴わないと「期待外れ」で離脱率激増
- 企業の公式コンテンツには不向き

## テクニック⑥: ストーリー風

### 心理的背景

**物語への共感**: 人間の脳は物語形式の情報を処理しやすく、記憶に残りやすい。特に「誰かの変化」「Before/After」はドラマ性を生み、読み手の感情移入を促す。

抽象的な情報より「具体的な人物の体験」の方が信頼性が高いと感じられる。

### プロンプトテンプレート

```
「〇〇」についてのキャッチコピーを10個作ってください。
小さな物語やドラマを感じさせるコピーにしてください。
「誰かの変化」「ビフォーアフター」「体験談」のような構造にしてください。
```

### 出力例の比較

| 通常の出力 | テクニック適用後 |
|-----------|----------------|
| 転職を成功させる3つの準備 | ブラック企業で消耗してた私が、ホワイト企業に転職できた理由 |
| 効果的な転職活動のコツ | 『もう辞めたい』と思った翌週、まさか運命の会社に出会うなんて |
| 転職エージェントの選び方 | 7社落ちた私を救った、たった1人のエージェントの言葉 |

### 効果的な使用場面

- 悩み解決系（ダイエット、転職、恋愛等）
- Before/Afterが明確なサービス
- ランディングページの導入部分

### 注意点

- 嘘のストーリーは絶対に禁止（薬機法・景品表示法違反のリスク）
- 長すぎるとタイトルとして機能しない
- 「誰の話?」が曖昧だと共感が得られない

## テクニック⑦: ターゲット限定

### 心理的背景

**特別感・所属欲求**: 「これは自分のためのものだ」と感じると、人は強く反応する。逆に万人向けのメッセージは誰にも刺さらない。

ターゲットを極限まで絞ることで、「他の人は無視していいから、あなただけに伝えたい」という特別感を演出できる。

### プロンプトテンプレート

```
「〇〇」についてのキャッチコピーを10個作ってください。
ターゲットを極限まで絞り込んでください。
「〇〇な人だけ」「〇〇したことがある人へ」のように、特定の人にだけ響くコピーにしてください。
```

### 出力例の比較

| 通常の出力 | テクニック適用後 |
|-----------|----------------|
| 子育てを楽にする工夫 | イヤイヤ期で毎日泣いてるママだけ読んで |
| 育児のストレス解消法 | ワンオペ育児で限界を感じてる人へ |
| 子育て中の時短テクニック | 『もう無理』が口癖になってるママへ。5分でできること |

### 効果的な使用場面

- ニッチ市場（具体的な悩みを持つ層）
- 競合が多いジャンル（差別化したいとき）
- コミュニティ形成（「同じ悩みを持つ仲間」感）

### 注意点

- 絞りすぎると母数が減る（ビジネス的にはバランスが重要）
- ターゲット外の人に「排除された」印象を与える可能性
- 限定感が嘘だとすぐバレる

## テクニック⑧: 失敗談

### 心理的背景

**親近感・信頼性**: 完璧な成功談より「失敗からの逆転」の方が共感を得やすい。読み手は「自分と同じ失敗をした人」に親近感を持ち、その解決策を信頼する。

また、失敗を公開することで「正直さ」「誠実さ」が伝わり、権威性が増す。

### プロンプトテンプレート

```
「〇〇」についてのキャッチコピーを10個作ってください。
失敗談や恥ずかしい体験を含めてください。
「〇〇で失敗した」「〇〇で恥をかいた」のような、人間味のあるコピーにしてください。
失敗や恥ずかしい体験のあと、今ではそれが解決されていることが伝わる感じで。
```

### 出力例の比較

| 通常の出力 | テクニック適用後 |
|-----------|----------------|
| 私にもできたプロの味。オンライン料理教室初月980円。 | 肉じゃがを『煮物?』と聞かれた日、私はここに申し込んだ。 |
| 家族に喜ばれる料理が作れる | 『給食の方がおいしい』って言われた夜、この本を買った。今は息子がおかわりしてくる。 |
| 初心者でも安心の料理講座 | 卵焼きすら焦がしてた私が、今では『レシピ教えて』って言われる |

### 効果的な使用場面

- 初心者向けサービス（ハードルを下げる）
- 悩み解決系（「自分も同じだった」で共感獲得）
- Before/Afterが明確な商品

### 注意点

- 失敗が軽すぎると共感が得られない（「1回だけ失敗」等）
- 失敗だけで終わると暗い印象
- 嘘の失敗談は絶対に禁止

## テクニック⑨: 比較構造

### 心理的背景

**選択の錯覚**: 人は選択肢を与えられると判断しやすくなる。「AかB」という構造は、読み手に「どっちだろう?」と考えさせ、記事への関心を高める。

また、比較形式は「優劣」「メリット/デメリット」を明示でき、意思決定を促しやすい。

### プロンプトテンプレート

```
「〇〇」についてのキャッチコピーを10個作ってください。
「AかB」「AよりB」のような比較構造にしてください。
読み手が「どっちだろう?」と気になるようなコピーにしてください。
```

### 出力例の比較

| 通常の出力 | テクニック適用後 |
|-----------|----------------|
| 効果的な筋トレ方法 | 『自宅トレ』vs『ジム通い』、2026年の正解はどっち? |
| 筋トレの頻度について | 毎日やる?それとも週3回に絞る? |
| おすすめ筋トレグッズ | ダンベルかケトルベルか。プロが選ぶのは… |

### 効果的な使用場面

- 迷いがちなテーマ（選択肢が複数ある）
- 比較記事、レビュー記事
- 議論を呼びたいとき（SNSでのエンゲージメント狙い）

### 注意点

- どちらかに偏りすぎると「ステマ?」と疑われる
- 比較対象が不適切だと意味がない（「リンゴとパソコン」等）
- 結論を明示しないと読み手がモヤモヤする

## テクニック⑩: 秘密・裏技

### 心理的背景

**希少性の原理**: 「他の人は知らない情報」は価値が高いと感じられる。特に「先駆者が隠している」「プロが教えたがらない」等のフレーズは、読み手に「特別な情報にアクセスできる」優越感を与える。

情報商材マーケティングで頻繁に使われる手法。

### プロンプトテンプレート

```
「〇〇」についてのキャッチコピーを10個作ってください。
「秘密」「裏技」「隠された」「誰も教えてくれない」のような、特別な情報があることを示すコピーにしてください。
```

### 出力例の比較

| 通常の出力 | テクニック適用後 |
|-----------|----------------|
| 民泊で稼ぐコツ | 先駆者たちが隠している、民泊で稼ぐコツ |
| 物件選びのポイント | プロが教えたがらない物件選びの裏技 |
| 民泊運営の始め方 | 誰も教えてくれない、民泊で月30万円稼ぐ設計図 |

### 効果的な使用場面

- ノウハウ系、情報商材
- ニッチな専門知識
- 競合との差別化

### 注意点

- 中身が「秘密」でも何でもないと信頼を失う
- 「隠す」理由が不明だと不自然（なぜプロは隠すのか?）
- やりすぎると陰謀論っぽくなる

## テクニック⑪: 時間軸・緊急性

### 心理的背景

**損失回避バイアス**: 人は「何かを得る喜び」より「何かを失う痛み」を2倍強く感じる。「今やらないと手遅れ」というメッセージは、行動を強力に促す。

ただし、毎回緊急性を訴えると「オオカミ少年」と化し、信頼を失う。

### プロンプトテンプレート

```
「〇〇」についてのキャッチコピーを10個作ってください。
「今すぐ」「まだ間に合う」「手遅れになる前に」のような、時間的な緊急性を感じさせるコピーにしてください。
```

### 出力例の比較

| 通常の出力 | テクニック適用後 |
|-----------|----------------|
| 絶対に合格したい人のための資格勉強法 | その勉強法、今すぐ見直して。試験まで間に合わなくなる |
| 合格率を上げる記述テクニック | まだ間に合う。今すぐこの自由記述テクニックを覚えて |
| 効率的な暗記法 | あと2週間で手遅れ。今日から変えるべき暗記法 |

### 効果的な使用場面

- 期限があるもの（試験、締切、季節商品等）
- セール、キャンペーン告知
- ランディングページのCTA（行動喚起）

### 注意点

- 嘘の緊急性は景品表示法違反のリスク
- 毎回使うと「またか」と無視される
- 緊急性の根拠が不明だと不信感

## テクニック⑫: 擬音語

### 心理的背景

**シズル感・臨場感**: 擬音語・擬態語は五感に訴え、無機質な情報を「体感」に変える。特に食品、美容、体験型サービスとの相性が良い。

「ツルツル」「ザクザク」「スッキリ」等は、結果のイメージを具体化し、欲求を刺激する。

### プロンプトテンプレート

```
「〇〇」についてのキャッチコピーを10個作ってください。
擬音語・擬態語を必ず入れてください。
「ドカン」「ガラッと」「ザクザク」「スルスル」のように、臨場感・シズル感のあるコピーにしてください。
```

### 出力例の比較

| 通常の出力 | テクニック適用後 |
|-----------|----------------|
| 塗って流すだけ。簡単ムダ毛ケア | 塗って、流したら、ツルン。触るたびに感動する肌へ |
| 家計簿アプリでお金の管理 | ザクザク貯まる、パッと見える。お金の不安がスーッと消える家計簿アプリ |
| 疲労回復サプリメント | グッタリがシャキーン!翌朝スッキリ、仕事サクサク |

### 効果的な使用場面

- 食品、飲料（「ジュワ〜」「モチモチ」等）
- 美容、健康（「ツルツル」「スッキリ」等）
- 体験型サービス（「ワクワク」「ドキドキ」等）

### 注意点

- B2B、専門性の高いコンテンツには不向き
- 使いすぎると安っぽい印象
- 擬音語が商品/サービスの実態と合わないと違和感

## テクニック⑬: 自虐

### 心理的背景

**ハードル下げ**: 完璧主義を避け、「自分はダメだけど」と前置きすることで、読み手に「この人も自分と同じだ」という親近感を与える。

特に初心者向けサービスでは、「センスない」「運動音痴」等の自虐が「それでも大丈夫」という安心感につながる。

### プロンプトテンプレート

```
「〇〇」についてのキャッチコピーを10個作ってください。
自虐的なトーンにしてください。
「自分はダメだけど」「センスないけど」のような、謙遜したコピーにしてください。
自虐した後、今ではそれが解決されていることが伝わる感じで。
```

### 出力例の比較

### 効果的な使用場面

- 初心者向けサービス
- ハードルが高いと思われがちな分野（プログラミング、投資等）
- 親しみやすさを重視したいとき

### 注意点

- 自虐だけで終わると暗い印象
- プロフェッショナルなイメージを損なう場合がある
- B2B、高級ブランドには不向き

## テクニック⑭: データ風

### 心理的背景

**権威への服従**: 人は「科学的根拠」「統計データ」に弱い。「調査でわかった」「データが示す」等のフレーズは、情報の信憑性を高め、説得力を増す。

ただし、架空データや誇張は絶対に禁止。信頼を失うだけでなく、法的リスクもある。

### プロンプトテンプレート

```
「〇〇」についてのキャッチコピーを10個作ってください。
統計データや調査結果があるような書き方にしてください。
「〇〇%が」「調査でわかった」「データが示す」のような、信憑性を感じさせるコピーにしてください。
```

### 出力例の比較

| 通常の出力 | テクニック適用後 |
|-----------|----------------|
| 読書量が増える習慣づくり | 調査で判明。読書量が増える人には、ある共通の習慣があった |
| 人生が変わる本の選び方 | 500人を分析してわかった、人生が変わる本の選び方3原則 |
| 効果的な読書法 | データが示す『読んだつもり』を防ぐ科学的読書法 |

### 効果的な使用場面

- B2B、教育系コンテンツ
- 専門性をアピールしたいとき
- 論理的説得が必要な場面

### 注意点

- **架空データは絶対に禁止**（景品表示法・不正競争防止法違反）
- データの出典を明示できない場合は使わない
- 「データ風」だけで中身がないと信頼を失う

## テクニック⑮: 呼びかけ

### 心理的背景

**対話的関係性**: 「あなた」「君」「みんな」等の呼びかけは、一方通行の情報発信を双方向のコミュニケーションに変える。

読み手は「自分に話しかけられている」と感じ、親密さや関心が高まる。特にSNS、ブログでの効果が高い。

### プロンプトテンプレート

```
「〇〇」についてのキャッチコピーを10個作ってください。
読み手に直接語りかける口調にしてください。
「あなた」「君」「みんな」のように、親しく呼びかけるコピーにしてください。
```

### 出力例の比較

| 通常の出力 | テクニック適用後 |
|-----------|----------------|
| 効率的な時間の使い方 | ねえ、あなたのその時間の使い方、もったいなさすぎる |
| タイムマネジメント術 | 君のタイムマネジメント、実はめちゃくちゃポテンシャルあるのに |
| 時間を有効活用する方法 | みんな気づいてないけど、その5分が人生変えるって話 |

### 効果的な使用場面

- SNS（Twitter、Instagram等）
- カジュアルなブログ
- 若年層向けコンテンツ

### 注意点

- フォーマルな場面（企業サイト、公式リリース等）には不向き
- 距離感を間違えると馴れ馴れしい印象
- ターゲット層によって呼びかけ方を変える必要がある（「あなた」「君」「お前」等）

## まとめ: プロンプト活用のポイント

1. **単体より組み合わせ**: 2〜3個のテクニックを組み合わせると効果倍増
2. **中身との整合性**: コピーで上げた期待値は本文で必ず満たす
3. **やりすぎ注意**: 感情ワード過多、緊急性の乱用は信頼を損なう
4. **媒体に合わせる**: SNS（カジュアル）vs 企業サイト（フォーマル）
5. **ターゲット確認**: 初心者向け（自虐、失敗談）vs 専門家向け（データ、数字）
6. **架空データ禁止**: テクニック⑭は特に注意。法的リスクあり

詳細な使い方は `SKILL.md` を参照してください。
